<article><div class="article_section">
          <h1>IObit forums hacked to spread ransomware to its members</h1>
          <div class="cz-news-story-title-section">
            <div class="cz-news-title-left-area">
                            By <h6><a rel="author" href="https://www.bleepingcomputer.com/author/lawrence-abrams/" class="author"><span itemprop="author" itemscope itemtype="https://schema.org/Person"><span itemprop="name">Lawrence Abrams</span></span></a></h6>
            </div>
	    <meta itemprop="articleSection" content="Security"><div class="cz-news-title-right-area">
              <ul><li class="cz-news-date">January 18, 2021</li>
                <li class="cz-news-time">02:57 PM</li>
                
                <li class="cz-news-comment"><a href="#comments">6</a></li>
              </ul></div>
          </div>
	  <div class="articleBody">
	   
<p style="text-align:center"><img alt="Iobit" height="800" src="https://www.bleepstatic.com/content/hl-images/2021/01/18/IObit-logo.jpg" width="1602"></p>

<p>Windows utility developer IObit was hacked over the weekend to perform a widespread attack to distribute the strange DeroHE ransomware to its forum members.</p>

<p>IObit is a software developer known for Windows system optimization and anti-malware programs, such as Advanced SystemCare.</p>

<p>Over the weekend, IObit forum members began receiving emails claiming to be from IObit stating that they are entitled to a free 1-year license to their software as a special perk of being a forum member.</p>

<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="IObit Promo email" height="600" src="https://www.bleepstatic.com/images/news/ransomware/d/derohe-iobit/iobit-email.jpg" width="878"><figcaption><strong>IObit 'Promo' email</strong></figcaption></figure></div>

<p>Included in the email is a 'GET IT NOW' link&nbsp;that redirects to hxxps://forums.iobit.com/promo.html. This page no longer exists,&nbsp;but at the time of the attack,&nbsp;it was distributing a file at hxxps://forums.iobit.com/free-iobit-license-promo.zip.</p>

<p>This zip file [<a href="https://www.virustotal.com/gui/file/2138091055ad48988e5b94a6ca95663ef715dbd36893e59d71269318bcf7aeb5/detection" target="_blank" rel="nofollow noopener">VirusTotal</a>] contains digitally signed files from the legitimate IObit&nbsp;License Manager program, but with the IObitUnlocker.dll replaced with an unsigned&nbsp;malicious version shown below.</p>

<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="Malicious IObitUnlocker.dll DLL" height="500" src="https://www.bleepstatic.com/images/news/ransomware/d/derohe-iobit/dll-hijacking.jpg" width="683"><figcaption><strong>Malicious IObitUnlocker.dll DLL</strong><br>
Source: BleepingComputer</figcaption></figure></div>

<p>When IObit License Manager.exe is executed, the malicious IObitUnlocker.dll will be executed to install the DeroHE ransomware to&nbsp;C:\Program Files (x86)\IObit\iobit.dll [<a href="https://www.virustotal.com/gui/file/976af19ce19cd9dc4ff6fd7cb580c16fac25c046ad9fd529bf50451db6032727/detection" target="_blank" rel="nofollow noopener">VirusTotal</a>]and execute it.</p>

<p>As most executables are signed with IOBit's certificate, and the zip file was hosted on their site,&nbsp;<a href="https://www.bleepingcomputer.com/forums/t/741190/derohe-ransomware-distributed-through-fake-iobit-one-year-free-license-key-promo/" target="_blank" rel="nofollow noopener">users installed the ransomware</a>&nbsp;thinking it was a legitimate promotion.</p>

<p>Based on reports at IObit's forum&nbsp;and other forums [<a href="https://malwaretips.com/threads/iobit-forum-hacked.106312/" target="_blank" rel="nofollow noopener">1</a>, <a href="https://www.wilderssecurity.com/threads/iobit-forums-hacked.435960/" target="_blank" rel="nofollow noopener">2</a>], this is a widespread attack that targeted all forum members.</p>

<h2>A closer look at the DeroHE ransomware</h2>

<p>BleepingComputer has since analyzed the ransomware to illustrate what happens when executed on a victim's computer.</p>

<p>When first started, the ransomware will add a Windows autorun named "IObit License Manager" that launches&nbsp;the "rundll32 "C:\Program Files (x86)\IObit\iobit.dll",DllEntry" command when logging in to Windows.</p>

<p>Emsisoft analyst&nbsp;<a href="https://twitter.com/themalwarequeen" target="_blank" rel="nofollow noopener">Elise van Dorp</a>, who also analyzed the ransomware, stated the ransomware adds the following Windows Defender exclusions to allow the DLL to run.</p>

<pre>
<code>@WMIC /Namespace:\\root\Microsoft\Windows\Defender class MSFT_MpPreference call Add ExclusionPath=\"
@WMIC /Namespace:\\root\Microsoft\Windows\Defender class MSFT_MpPreference call Add ExclusionPath=\"\Temp\\"
@WMIC /Namespace:\\root\Microsoft\Windows\Defender class MSFT_MpPreference call Add ExclusionExtension=\".dll\"
@WMIC /Namespace:\\root\Microsoft\Windows\Defender class MSFT_MpPreference call Add ExclusionProcess=\"rundll32.exe\"
</code></pre>

<p>The ransomware will now display a message box claiming to be from IObit License Manager stating, "Please wait. It may take a little longer than expected. Keep your computer running or screen on!' The ransomware shows this alert to prevent victims from shutting off their devices before the ransomware finishes.</p>

<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="Fake alert to not turn off the computer" height="220" src="data:image/gif;base64,R0lGODlhAQABAAAAACH5BAEKAAEALAAAAAABAAEAAAICTAEAOw==" width="566" data-src="https://www.bleepstatic.com/images/news/ransomware/d/derohe-iobit/alert.jpg" class="b-lazy"><figcaption><strong>Fake alert to not turn off the computer</strong><br>
Source: BleepingComputer</figcaption></figure></div>

<p>When encrypting victims, it will append the .DeroHE extension to encrypted files.</p>

<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="Files encrypted by the DeroHE ransomware" height="450" src="data:image/gif;base64,R0lGODlhAQABAAAAACH5BAEKAAEALAAAAAABAAEAAAICTAEAOw==" width="719" data-src="https://www.bleepstatic.com/images/news/ransomware/d/derohe-iobit/encrypted-files.jpg" class="b-lazy"><figcaption><strong>Files encrypted by the DeroHE ransomware</strong><br>
Source: BleepingComputer</figcaption></figure></div>

<p>Each encrypted file will also have a string of information appended to the end of the file, as shown below. The ransomware may use this information to decrypt files if a ransom is paid.</p>

<pre>
<code>{"version":"3","id":"dERiqiUutvp35oSUfRSTCXL53TRakECSGQVQ2hhUjuCEjC6zSNFZsRqavVVSdyEzaViULtCRPxzRwRCKZ2j2ugCg5r9SrERKe7r5DVpU8kMDr","parts":[{"size":193536,"esize":193564,"offset":0,"rm":"Phj8vfOREkYPKA9e9qke1EIYOGGciqkQBSzfzg=="}],"ext":".png"}
</code></pre>

<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="Hex edit of an encrypted file" height="600" src="data:image/gif;base64,R0lGODlhAQABAAAAACH5BAEKAAEALAAAAAABAAEAAAICTAEAOw==" width="682" data-src="https://www.bleepstatic.com/images/news/ransomware/d/derohe-iobit/hex-edit.jpg" class="b-lazy"><figcaption><strong>Hex edit of an encrypted file</strong><br>
Source: BleepingComputer</figcaption></figure></div>

<p>On the Windows desktop, the DeroHE ransomware will create two files named&nbsp;FILES_ENCRYPTED.html, containing a list of all encrypted files,&nbsp;and the READ_TO_DECRYPT.html ransom note.</p>

<p>The ransom note has the title of 'Dero Homomorphic Encryption,' and promotes a cryptocurrency called DERO. This note tells the victim to send 200 coins, worth approximately $100, to the listed address to get a decryptor.</p>

<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="DeroHE ransomware ransom note" height="600" src="data:image/gif;base64,R0lGODlhAQABAAAAACH5BAEKAAEALAAAAAABAAEAAAICTAEAOw==" width="865" data-src="https://www.bleepstatic.com/images/news/ransomware/d/derohe-iobit/ransom-note.jpg" class="b-lazy"><figcaption><strong>DeroHE ransomware ransom note</strong><br>
Source: BleepingComputer</figcaption></figure></div>

<p>Enclosed in the ransom note is the ransomware's&nbsp;Tor site&nbsp;http://deropayysnkrl5xu7ic5fdprz5ixgdwy6ikxe2g3mh2erikudscrkpqd.onion, which can be used to make the payment.</p>

<p>Of particular interest, the Tor site states that IObit can send $100,000 in DERO coins to decrypt all victims, as the attackers blame IObit for the compromise.</p>

<p>"Tell iobit.com to send us 100000 (1 hundred thousand) DERO coin to this address. dERopYDgpD235oSUfRSTCXL53TRakECSGQVQ2hhUjuCEjC6zSNFZsRqavVVSdyEzaViULtCRPxzRwRCKZ2j2ugCg26hRtLziwu"</p>

<p>"After payment arrive, all encrypted computer (including yours) will be decrypted. THIS IS IOBIT's FAULT to made your computer getting infected," the DeroHE Tor payment site states.</p>

<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="Dero Ransomware Tor payment site" height="600" src="data:image/gif;base64,R0lGODlhAQABAAAAACH5BAEKAAEALAAAAAABAAEAAAICTAEAOw==" width="691" data-src="https://www.bleepstatic.com/images/news/ransomware/d/derohe-iobit/tor-site.jpg" class="b-lazy"><figcaption><strong>Dero Ransomware Tor payment site</strong><br>
Source: BleepingComputer</figcaption></figure></div>

<p>The ransomware is being analyzed for weaknesses, and it is not known if it can be decrypted for free.</p>

<p>Furthermore, it is unknown if the threat actors will keep their word and provide a decryptor if payment is made.</p>

<h2>IObit forums likely compromised</h2>

<p>To create the fake promotion page and host a malicious download, the attackers likely hacked IObit's forum&nbsp;and gained access to an administrative account.</p>

<p>At this time, the forums still appear to be compromised, as if you visit missing pages that return a 404 error code, the web page will display dialogs to subscribe to browser notifications. Your browser will begin to receive desktop notifications promoting adult sites, malicious software, and other unwanted content when subscribed.</p>

<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="Compromised IObit forum page" height="450" src="data:image/gif;base64,R0lGODlhAQABAAAAACH5BAEKAAEALAAAAAABAAEAAAICTAEAOw==" width="916" data-src="https://www.bleepstatic.com/images/news/ransomware/d/derohe-iobit/hacked-forum.jpg" class="b-lazy"><figcaption><strong>Compromised IObit forum page</strong><br>
Source: BleepingComputer</figcaption></figure></div>

<p>Furthermore, if you click anywhere on the page, a new tab will open showing advertisements for adult sites. Other site sections also appear to be compromised as clicking on forum links redirect you to similar adult pages.</p>

<p>Attackers compromised the forum by injecting a malicious script on all pages that are not found, as shown below.</p>

<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="Compromised IObit forum page" height="282" src="data:image/gif;base64,R0lGODlhAQABAAAAACH5BAEKAAEALAAAAAABAAEAAAICTAEAOw==" width="1204" data-src="https://www.bleepstatic.com/images/news/ransomware/d/derohe-iobit/forum-comrpomise.jpg" class="b-lazy"><figcaption><strong>Compromised IObit forum page</strong><br>
Source: BleepingComputer</figcaption></figure></div>

<p>BleepingComputer has reached out to IObit with questions related to this attack but has not heard back.</p>

<p><strong><em>Updated 01/19/20: </em></strong>A security researcher known as <a href="https://twitter.com/RonnyTNL" target="_blank" rel="nofollow noopener">Ronny</a> told BleepingComputer IOBit is using vBulletin 5.6.1 for their forum software.</p>

<p>This version of vBulletin has a <a href="https://www.acunetix.com/vulnerabilities/web/vbulletin-5-6-1-nodeid-sql-injection/" target="_blank" rel="nofollow noopener">known vulnerabily</a> that allows remote attackers to gain control over the forum.</p>

function cz_strip_tags(input, allowed) {
  allowed = (((allowed || '') + '')
    .toLowerCase()
    .match(/<[a-z][a-z0-9]*>/g) || [])
    .join(''); // making sure the allowed arg is a string containing only tags in lowercase (<a><b><c>)
  var tags = /<\/?([a-z][a-z0-9]*)\b[^>]*>/gi,
    commentsAndPhpTags = /<!--[\s\S]*?-->|<\?(?:php)?[\s\S]*?\?>/gi;
  return input.replace(commentsAndPhpTags, '')
    .replace(tags, function($0, $1) {
      return allowed.indexOf('<' + $1.toLowerCase() + '>') > -1 ? $0 : '';
    });
}
function cz_br2nl(str) {
	var regex = /<br\s*[\/]?>/gi;
    //var pure_str = str.replace(regex,"\n");
    var pure_str = str.replace(regex,"");
    return cz_strip_tags(pure_str,'');
}
