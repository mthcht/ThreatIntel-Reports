          <h1>New Azer CryptoMix Ransomware Variant Released</h1>
          <div class="cz-news-story-title-section">
            <div class="cz-news-title-left-area">
                            By <h6><a rel='author' href='https://www.bleepingcomputer.com/author/lawrence-abrams/' class='author'><span itemprop="author" itemscope itemtype="https://schema.org/Person"><span itemprop="name">Lawrence Abrams</span></span></a></h6>
            </div>
	    <meta itemprop="articleSection" content="Security">
            <div class="cz-news-title-right-area">
              <ul>
                <li class="cz-news-date">July 5, 2017</li>
                <li class="cz-news-time">05:15 PM</li>
                
                <li class="cz-news-comment"><a href="#comment_form">0</a></li>
              </ul>
            </div>
          </div>
	  <div class="articleBody">
	   
<p>Today has been busy with ransomware&nbsp;and we have some some good news coming later today. For this story, though, we are going to take a look at the Azer variant of the Cryptomix ransomware. This version of Cryptomix&nbsp;was discovered today by security researcher <a href="https://twitter.com/malwrhunterteam/status/882562233628336129" rel="nofollow" target="_blank">MalwareHunterTeam</a>&nbsp;right as a <a href="https://www.bleepingcomputer.com/news/security/decryptor-released-for-the-mole02-cryptomix-ransomware-variant/" target="_blank" rel="nofollow noopener">decryptor for the previous version</a>, Mole02, was released.</p>

<p>While this ransomware encrypts files in a similar manner to all others in this family, I did notice some changes in this version that will be outlined below.</p>

<p>As we are always looking for weaknesses, if you are a victim of this variant and decide to pay the ransom, please <a href="https://www.bleepingcomputer.com/submit-malware.php?channel=168" target="_blank" rel="nofollow noopener">send us the decryptor</a> so we can take a look at it. You can also discuss or receive support for Cryptomix ransomware infections in our dedicated <a href="https://www.bleepingcomputer.com/forums/t/611907/cryptomix-or-crypmix-ransomware-help-topic-revenge-cryptoshield-extensions/" target="_blank" rel="nofollow noopener">Cryptomix&nbsp;Help &amp; Support Topic</a>.</p>

<h3>Changes in the Azer&nbsp;Cryptomix Ransomware Variant</h3>

<p>While overall the encryption methods stay the same in this variant, there have been some differences. First and foremost, we have a new ransom note with a file name of&nbsp;<strong>_INTERESTING_INFORMACION_FOR_DECRYPT.TXT</strong>. This ransom note contains instructions to contact either <strong>webmafia@asia.com</strong> or&nbsp;<strong>donald@trampo.info</strong> for payment information.</p>

<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="Azer Ransom Note" height="278" src="https://www.bleepstatic.com/images/news/ransomware/c/cryptomix/azer/ransom-note.png" width="852"><figcaption><strong>Azer Ransom Note</strong></figcaption></figure></div>

<p>The next noticeable change is the extension appended to encrypted files. With this version, when a file is encrypted by the ransomware, it will modify the filename and then append the string&nbsp;<strong>-email-[email_address].AZER</strong> to the encrypted file. For example, an test&nbsp;file encrypted by this variant has an encrypted file name of&nbsp;32A1CD301F2322B032AA8C8625EC0768-email-[webmafia@asia.com].AZER.&nbsp;</p>

<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="Folder of Encrypted Azer Files" height="524" src="https://www.bleepstatic.com/images/news/ransomware/c/cryptomix/azer/encrypted-files.png" width="879"><figcaption><strong>Folder of Encrypted Azer Files</strong></figcaption></figure></div>

<p>Last, but not least, this version performs no network communication and is completely offline. It also embeds ten different RSA-1024 public encryption keys, which are listed below. One of these&nbsp;keys will be selected to encrypt the AES key&nbsp;used to encrypt a victim's files. This is quite different compared to the Mole02 variant, which only included one public RSA-1024 key.</p>

<p>As this is just a cursory analysis of this new variant, if anything else is discovered, we will be sure to update this article.<br>
&nbsp;</p>

function cz_strip_tags(input, allowed) {
  allowed = (((allowed || '') + '')
    .toLowerCase()
    .match(/<[a-z][a-z0-9]*>/g) || [])
    .join(''); // making sure the allowed arg is a string containing only tags in lowercase (<a><b><c>)
  var tags = /<\/?([a-z][a-z0-9]*)\b[^>]*>/gi,
    commentsAndPhpTags = /<!--[\s\S]*?-->|<\?(?:php)?[\s\S]*?\?>/gi;
  return input.replace(commentsAndPhpTags, '')
    .replace(tags, function($0, $1) {
      return allowed.indexOf('<' + $1.toLowerCase() + '>') > -1 ? $0 : '';
    });
}
function cz_br2nl(str) {
	var regex = /<br\s*[\/]?>/gi;
    //var pure_str = str.replace(regex,"\n");
    var pure_str = str.replace(regex,"");
    return cz_strip_tags(pure_str,'');
}
