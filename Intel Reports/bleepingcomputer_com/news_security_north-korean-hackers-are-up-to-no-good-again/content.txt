<article><div class="article_section">
          <h1>North Korean Hackers Are up to No Good Again</h1>
          <div class="cz-news-story-title-section">
            <div class="cz-news-title-left-area">
                            By <h6><a rel="author" href="https://www.bleepingcomputer.com/author/catalin-cimpanu/" class="author"><span itemprop="author" itemscope itemtype="https://schema.org/Person"><span itemprop="name">Catalin Cimpanu</span></span></a></h6>
            </div>
	    <meta itemprop="articleSection" content="Security"><div class="cz-news-title-right-area">
              <ul><li class="cz-news-date">April 27, 2018</li>
                <li class="cz-news-time">06:05 AM</li>
                
                <li class="cz-news-comment"><a href="#comments">2</a></li>
              </ul></div>
          </div>
	  <div class="articleBody">
	   
<p style="text-align:center"><img alt="GhostSecret" height="455" src="https://www.bleepstatic.com/content/posts/2018/04/27/GhostSecret.png" width="1250"></p>


<p>For a month leading up to <a href="http://abcnews.go.com/International/north-koreas-kim-jong-crosses-dmz-historic-meeting/story?id=54759591" rel="nofollow" target="_blank">today's historic meet</a> between North and South Korea's presidents, a North Korean hacking group has amplified operations and has targeted a wide variety of business sectors in at least 17 countries.</p>


<p>The purpose of this campaign was to infect organizations, perform reconnaissance, and steal sensitive data. Targeted industries included critical infrastructure, entertainment, finance, healthcare, and telecommunications.</p>


<p>This overly aggressive campaign appears to be a new operation carried out by a group of hackers primarily known as Lazarus Group, the ones responsible for the infamous Sony Studios hack in 2014. <a href="https://docs.google.com/spreadsheets/d/1H9_xaxQHpWaa4O_Son4Gx0YOIzlcBWMsdvePFX68EKU/pubhtml" rel="nofollow" target="_blank">Other names</a> for this group are <a href="https://www.bleepingcomputer.com/news/security/dhs-and-fbi-publish-details-on-deltacharlie-north-koreas-ddos-botnet/" target="_blank" rel="nofollow noopener">Hidden Cobra</a>, the name US authorities are using to describe it, but also the Hastati Group, Group 77, or Labyrinth Chollima.</p>

<div align="center" style="width:98%; margin:0 auto; text-align:center; padding:4px; background:#f0f0f0; border:1px solid #ccc; border-radius:6px;">
 <a href="https://www.wiz.io/lp/secure-images-101-a-visual-guide?utm_source=bleepingcomputer&amp;utm_medium=display&amp;utm_campaign=FY26Q4_INB_FORM_Hardened-Images-101-Digital-Poster&amp;sfcid=701Py00000WFCeLIAX&amp;utm_term=FY27-bleepingcomputer-article-970x250&amp;utm_content=Secured-Images-101" rel="nofollow noopener" target="_blank"><img src="https://www.bleepstatic.com/c/w/Secured-Images-970x250.png" alt="Wiz" style="margin-top: 0px;"></a>
</div>


<p>The group operates in bursts of hacking activity aimed at specific targets. Past operations include <a href="https://cyber-peace.org/wp-content/uploads/2013/07/wp-operation-troy.pdf" rel="nofollow" target="_blank">Operation Troy</a>, <a href="https://www.operationblockbuster.com/wp-content/uploads/2016/02/Operation-Blockbuster-Report.pdf" rel="nofollow" target="_blank">Blockbuster</a>, or <a href="https://researchcenter.paloaltonetworks.com/2015/11/tdrop2-attacks-suggest-dark-seoul-attackers-return/" rel="nofollow" target="_blank">Dark Seoul</a>.</p>


<h2>Operation GhostSecret started last month</h2>

<p>The most recent Lazarus Group operation is codenamed Operation GhostSecret and appears to have started in mid-March 2018 with <a href="https://securingtomorrow.mcafee.com/mcafee-labs/hidden-cobra-targets-turkish-financial-sector-new-bankshot-implant/" rel="nofollow" target="_blank">attacks targeting the Turkish financial sector</a>.</p>


<p>Cybersecurity firm McAfee describes this particular campaign as sophisticated due to the "significant capabilities, demonstrated by their tools development and the pace at which [the attackers] operate."</p>


<p>Researchers identified malware with shared capabilities to hacking tools used in the 2014 Sony hack, but they also found newer tools such as Bankshot (implant), Proxysvc (downloader), and Escad (backdoor).</p>


<h2>Attacks amplify in scale following public disclosure</h2>

<p>Experts noted that despite exposing the first attacks on the Turkish financial sector, the group continued its attacks unphased by the attention their tools and hacking infrastructure were getting.</p>


<p>Furthermore, attacks seem to have ramped up, most likely in an attempt to use hacking tools while they were still effective and before security software would be able to detect them.</p>


<p>McAfee said it alerted the Thai government that some of the command and control servers used for Operation GhostSecret were hosted on the compromised servers of Thammasat University in Bangkok. ThaiCERT seized and shut down the servers <a href="https://www.thaicert.or.th/alerts/admin/2018/al2018ad001.html" target="_blank" rel="nofollow">two days ago</a>.</p>


<p><em>A full and detailed report on Operation GhostSecret, including IoCs, is available <a href="https://securingtomorrow.mcafee.com/mcafee-labs/analyzing-operation-ghostsecret-attack-seeks-to-steal-data-worldwide/" rel="nofollow" target="_blank">here</a>.</em></p>

function cz_strip_tags(input, allowed) {
  allowed = (((allowed || '') + '')
    .toLowerCase()
    .match(/<[a-z][a-z0-9]*>/g) || [])
    .join(''); // making sure the allowed arg is a string containing only tags in lowercase (<a><b><c>)
  var tags = /<\/?([a-z][a-z0-9]*)\b[^>]*>/gi,
    commentsAndPhpTags = /<!--[\s\S]*?-->|<\?(?:php)?[\s\S]*?\?>/gi;
  return input.replace(commentsAndPhpTags, '')
    .replace(tags, function($0, $1) {
      return allowed.indexOf('<' + $1.toLowerCase() + '>') > -1 ? $0 : '';
    });
}
function cz_br2nl(str) {
	var regex = /<br\s*[\/]?>/gi;
    //var pure_str = str.replace(regex,"\n");
    var pure_str = str.replace(regex,"");
    return cz_strip_tags(pure_str,'');
}

		if (cz_top_of_window > cz_header_pos) {

			$('.bc_goto_top').fadeOut("slow");			
		}
	});

});
var prevScrollTop = 0;
$(window).scroll(function(event){
    scrollTop = $(this).scrollTop();

    if ( scrollTop < 0 ) {
        scrollTop = 0;
    }
    if ( scrollTop > $('body').height() - $(window).height() ) {
        scrollTop = $('body').height() - $(window).height();
    }

    if (scrollTop >= prevScrollTop && scrollTop) {
        $('header').addClass("nav-up");
    } else {
	if (scrollTop >=100)
	{
	 $('header').removeClass("nav-up"); 
	 main_nav_hide_timer = setTimeout("call_main_nav_hide()",5000);
	}
	else
	{
	 $('header').removeClass("nav-up");
	 clearInterval(main_nav_hide_timer);
	}
    }

    prevScrollTop = scrollTop;
});
