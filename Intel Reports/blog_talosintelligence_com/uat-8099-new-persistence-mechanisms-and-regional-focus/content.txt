    <h1 class="text-center">Dissecting UAT-8099: New persistence mechanisms and regional focus</h1>

    <div class="text-center m-1">
        <div class="post-author">
            <span>By </span>
                <a href="https://blog.talosintelligence.com/author/joey/">Joey Chen</a>
        </div>

        <br/>
        <time class="post-datetime" datetime="January 29, 2026 06:00">

    <section class="post-content-wrapper mt-5">
        <div class="post-content" id="post-content">
            <ul><li>Cisco&nbsp;Talos has&nbsp;identified&nbsp;a new campaign by UAT-8099,&nbsp;active from late 2025 to early 2026,&nbsp;that is&nbsp;targeting&nbsp;vulnerable&nbsp;Internet Information Services&nbsp;(IIS)&nbsp;servers across Asia with a specific&nbsp;focus&nbsp;on victims in Thailand and Vietnam.&nbsp;</li><li>Analysis confirms&nbsp;significant&nbsp;operational overlaps between this activity and the&nbsp;<a href="https://labs.withsecure.com/publications/webjack" rel="noreferrer noopener"><u>WEBJACK</u></a>&nbsp;campaign. This includes&nbsp;critical indicators of compromise including malware hashes,&nbsp;command&nbsp;and control (C2), and victimology.&nbsp;</li><li>UAT-8099&nbsp;uses&nbsp;web&nbsp;shells and PowerShell to execute scripts and deploy the&nbsp;GotoHTTP&nbsp;tool, granting the&nbsp;threat&nbsp;actor remote access to&nbsp;vulnerable&nbsp;IIS servers.&nbsp;</li><li>New variants of&nbsp;BadIIS&nbsp;now hardcode the target region directly into the malware, offering customized features for each specific&nbsp;variant.&nbsp;These customizations include exclusive file extensions, corresponding dynamic page extensions, directory indexing configurations, and the ability to load HTML templates from local files.&nbsp;</li><li>A&nbsp;Linux&nbsp;Executable and Linkable Format (ELF)&nbsp;variant of&nbsp;BadIIS&nbsp;was uploaded to&nbsp;VirusTotal&nbsp;on&nbsp;Oct. 1,&nbsp;2025.&nbsp;The malware includes proxy mode, injector mode,&nbsp;and&nbsp;search engine optimization (SEO)&nbsp;fraud mode,&nbsp;similar to&nbsp;what Talos described&nbsp;in the&nbsp;<a href="https://blog.talosintelligence.com/uat-8099-chinese-speaking-cybercrime-group-seo-fraud/" rel="noreferrer noopener"><u>previous UAT-8099 blog</u></a>.</li></ul><hr><h2 id="uat-8099-new-activity">UAT-8099&nbsp;new activity&nbsp;</h2><p>Cisco Talos&nbsp;observed&nbsp;new activity from&nbsp;<a href="https://blog.talosintelligence.com/uat-8099-chinese-speaking-cybercrime-group-seo-fraud/" rel="noreferrer noopener"><u>UAT-8099</u></a>&nbsp;spanning from August 2025 through early 2026. Analysis of Cisco's file census and DNS traffic&nbsp;indicates&nbsp;that compromised IIS servers are&nbsp;located&nbsp;across India, Pakistan, Thailand, Vietnam, and Japan, with a distinct concentration of attacks in Thailand and Vietnam. Furthermore, this activity significantly overlaps with the&nbsp;<a href="https://labs.withsecure.com/publications/webjack" rel="noreferrer noopener"><u>WEBJACK</u></a>&nbsp;campaign; we have&nbsp;identified&nbsp;high-confidence correlations across malware hashes, C2 infrastructure, victimology, and the promoted gambling sites.</p><figure class="kg-card kg-image-card kg-card-hascaption"><img src="https://blog.talosintelligence.com/content/images/2026/01/Figure-1.png" class="kg-image" alt="" loading="lazy" width="1319" height="929" srcset="https://blog.talosintelligence.com/content/images/size/w600/2026/01/Figure-1.png 600w, https://blog.talosintelligence.com/content/images/size/w1000/2026/01/Figure-1.png 1000w, https://blog.talosintelligence.com/content/images/2026/01/Figure-1.png 1319w" sizes="(min-width: 720px) 720px"><figcaption><span style="white-space: pre-wrap;">Figure&nbsp;1.&nbsp;Content for crawlers.</span></figcaption></figure><p>While the threat actor continues to rely on web&nbsp;shells,&nbsp;SoftEther&nbsp;VPN, and&nbsp;EasyTier&nbsp;to control compromised IIS servers, their operational strategy has evolved significantly. First, this latest campaign marks a shift in their black hat SEO tactics toward a more specific regional focus. Second, the actor increasingly&nbsp;leverages&nbsp;red team&nbsp;utilities and legitimate tools to evade detection and&nbsp;maintain&nbsp;long-term persistence.</p><h3 id="infection-chain">Infection chain&nbsp;</h3><p>Upon gaining initial access, the&nbsp;threat&nbsp;actor executes standard reconnaissance commands, such as&nbsp;<code>whoami</code>&nbsp;and&nbsp;<code>tasklist</code>, to gather system&nbsp;information. Following this, they deploy VPN tools and&nbsp;establish&nbsp;persistence&nbsp;by creating a hidden user account named&nbsp;“admin$”.&nbsp;UAT-8099 has&nbsp;further&nbsp;expanded their arsenal with the several new tools below:&nbsp;</p><ul><li>Sharp4RemoveLog: A .NET utility designed to clear all Windows event logs, effectively erasing forensic traces&nbsp;</li><li><a href="https://www.52pojie.cn/thread-1397484-1-1.html" rel="noreferrer noopener">CnCrypt&nbsp;Protect</a>: A&nbsp;Chinese-language&nbsp;file-protection&nbsp;utility.&nbsp;In this intrusion activity, it is abused to hide malicious files and&nbsp;facilitate&nbsp;dynamic-link library (DLL)&nbsp;redirection. This tool has been linked to&nbsp;previous&nbsp;IIS attacks since 2024, including SEO&nbsp;fraud&nbsp;campaigns targeting&nbsp;<a href="https://sec.vnpt.vn/2024/09/part-2-hacker-thuc-hien-black-hat-seo-cac-trang-web-bat-hop-phap-bang-tan-cong-redirect-nhu-the-nao" rel="noreferrer noopener">Vietnam</a>&nbsp;and&nbsp;<a href="https://mp.weixin.qq.com/s/TCrtN94jIMg7PLpfGquAYw" rel="noreferrer noopener">China</a>, as well as the&nbsp;<a href="https://labs.withsecure.com/publications/webjack" rel="noreferrer noopener">WEBJACK</a>&nbsp;campaign.&nbsp;</li><li><a href="https://github.com/c1earyy/OpenArk64" rel="noreferrer noopener">OpenArk64</a>: An&nbsp;open source&nbsp;anti-rootkit.&nbsp;The threat actor uses its kernel-level access to&nbsp;terminate&nbsp;security product processes that are otherwise protected from deletion.&nbsp;</li><li><a href="https://gotohttp.com/" rel="noreferrer noopener">GotoHTTP</a>: An online&nbsp;remote control&nbsp;tool. The threat actor uses&nbsp;VBscript&nbsp;to deploy this tool and let them remote control the compromised server.&nbsp;Talos provides&nbsp;more&nbsp;detail&nbsp;in the following section.&nbsp;&nbsp;</li></ul><p>Subsequently, the threat actor deploys two archive files&nbsp;containing&nbsp;the latest version of the&nbsp;BadIIS&nbsp;malware.&nbsp;Notably, the file&nbsp;names of these archives are correlated with the specific geographic regions targeted by the&nbsp;BadIIS malware; for example,&nbsp;“VN”&nbsp;denotes Vietnam and&nbsp;“TH”&nbsp;denotes Thailand.</p>

  function coverageView() {
    const coverageTable = document.getElementById('coverage-block');
    const sidebar = document.getElementById('side-bar');
    const postContent = document.getElementById('post-content');
    const currentWidth = window.innerWidth;  
  
    if (coverageTable && sidebar && postContent) {
      if (currentWidth < 1000) {
        // For viewport less than 1000px, append to post content
        postContent.appendChild(coverageTable);
      } else {
        // For viewport 1000px and wider, check for subscribe button
        const subscribeButton = sidebar.querySelector('.subscribe-button');
        
        if (subscribeButton) {
          // Insert coverage block after the subscribe button
          subscribeButton.insertAdjacentElement('afterend', coverageTable);
        } else {
          // No subscribe button found, prepend to top of sidebar
          sidebar.prepend(coverageTable);
        }
      }
    }
  }
  
  window.addEventListener("resize", coverageView);
  window.addEventListener("load", coverageView);
</script>
