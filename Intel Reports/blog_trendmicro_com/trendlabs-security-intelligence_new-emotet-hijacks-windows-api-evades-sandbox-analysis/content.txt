	<h1 class="article-details__title">New EMOTET Hijacks a Windows API, Evades Sandbox</h1>
	<p class="article-details__description">We recently discovered that EMOTET has a new iteration with a few changes in its usual behavior and new routines that allow it to elude sandbox and malware analysis.</p>
	<p class="article-details__author-by">By: Rubio Wu
		
			<time class="article-details__date">November 15, 2017</time>
		
		
		<span>Read time:&nbsp;</span><span class="eta"></span> (<span class="words"></span> words)
	</p>

            <div class="richText">
	
    


	
		<div>
			<p>We discussed the <a href="http://blog.trendmicro.com/trendlabs-security-intelligence/emotet-returns-starts-spreading-via-spam-botnet/">re-emergence of banking malware EMOTET </a>in September and how it has adopted a wider scope since it wasn't picky about the industries it attacks. We recently discovered that EMOTET has a new iteration (detected as TSPY_EMOTET.SMD10) with a few changes in its usual behavior and new routines that allow it to elude sandbox and malware analysis.</p>
<p>Based on our findings, EMOTET’s dropper changed from using RunPE to exploiting CreateTimerQueueTimer. CreateTimerQueueTimer is a <a href="https://msdn.microsoft.com/en-us/library/windows/desktop/ff818516(v=vs.85).aspx" target="_blank">Windows application programming interface (API)</a> that creates a <a href="https://msdn.microsoft.com/en-us/library/windows/desktop/ms686796(v=vs.85).aspx" target="_blank">queue for timers</a>. These timers are lightweight objects that enable the selection of a callback function at a specified time. The original <a href="https://blogs.blackberry.com/en/2017/09/threat-spotlight-man1-malware-the-last-crusade" target="_blank">function</a> of the API is to be part of the process chain by creating a timer routine, but here, the callback function of the API becomes EMOTET’s actual payload. EMOTET seems to have traded RunPE for a Windows API because the exploitation of the former has become popular while the latter is lesser known, theoretically making it more difficult to detect by security scanners.</p>
<p style="text-align: center;"><a href="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2017/11/171114Emotet_01.jpg"><img src="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2017/11/171114Emotet_01.jpg" alt="Figure 1. A CreateTimerQueueTimer API document "/></a></p>
<p style="text-align: center;"><i>Figure 1. A CreateTimerQueueTimer API document (from <a href="https://msdn.microsoft.com/zh-tw/library/windows/desktop/ms682485(v=vs.85).aspx" target="_blank">CreateTimerQueueTimer function</a>)</i></p>
<p style="text-align: center;"><a href="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2017/11/171114Emotet_02.jpg"><img src="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2017/11/171114Emotet_02.jpg" alt="Figure 2. When the EMOTET dropper executes at Stage 4"/></a></p>
<p style="text-align: center;"><i>Figure 2. When the EMOTET dropper executes at Stage 4, the Stage 5 payload at 0x 0x428310 will be injected to CreateTimerQueueTimer.</i></p>
<p style="text-align: left;">This is not the first malware we've seen abusing CreateTimerQueueTimer. <a href="https://success.trendmicro.com/solution/1115383-emerging-threat-on-hancitor)" target="_blank">Hancitor</a>, a banking Trojan that dropped PONY and <a href="http://blog.trendmicro.com/trendlabs-security-intelligence/banking-malware-vawtrak-now-uses-malicious-macros-abuses-windows-powershell/">VAWTRAK</a>, also exploited the API in its dropper, which is a malicious macro document.</p>
<p style="text-align: left;"><i><b>Anti-Analysis and Anti-Sandbox Techniques</b></i></p>
<p style="text-align: left;">We also observed a new behavior in this variant, which is its anti-analysis technique. Some malware are designed to sleep for a period of time to avoid detection from malware analysis products. The analysis platform will change its sleep period to a very short time to scan for malicious activities. EMOTET’s anti-analysis technique involves checking when the scanner monitors activities to dodge detection. CreateTimerQueueTimer helps EMOTET do the job every 0x3E8 milliseconds.</p>
<p style="text-align: left;">This variant has the ability to check if it’s inside a sandbox environment at the second stage of its payload. The EMOTET loader will not proceed if it sees that it’s running inside a sandbox environment.</p>
<p style="text-align: left;">The dropper will check for the following to discern whether it is running in a sandbox environment:</p>
<ul>
<li><span class="rte-red-bullet">When NetBIOS’ name is TEQUILABOOMBOOM.</span></li>
<li><span class="rte-red-bullet">When UserName is Wilber, NetBIOS’ name starts with SC, and NetBIOS name starts with CW.</span></li>
<li><span class="rte-red-bullet">When UserName is admin, DnsHostName is SystemIT, and if there’s a Debugger symbol file like C:\\Symbols\aagmmc.pdb.</span></li>
<li><span class="rte-red-bullet">When Username is admin, and NetBIOS name is KLONE_X64-PC</span></li>
<li><span class="rte-red-bullet">When UserName is John Doe.</span></li>
<li><span class="rte-red-bullet">When UserName is John and there are two files called C:\\take_screenshot.ps1 and C:\\loaddll.exe.</span></li>
<li><span class="rte-red-bullet">When these files are present: C:\\email.doc, C:\\123\\email.doc, and C:\\123\\\email.docx.</span></li>
<li><span class="rte-red-bullet">When these files are present: C:\\a\\foobar.bmp, C:\\a\\foobar.doc, and C:\\a\\foobar.gif.</span></li>
</ul>
<p style="text-align: center;"><a href="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2017/11/171114Emotet_03.jpg"><img src="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2017/11/171114Emotet_03.jpg" alt="Figure 3. When sample files are named sample., mlwr_smple. or artifact.exe, the malicious payload will also not be launched."/></a></p>
<p style="text-align: center;"><i>Figure 3. When sample files are named sample., mlwr_smple. or artifact.exe, the malicious payload will also not be launched.</i></p>
<p>As part of its unpacking technique, this variant will run itself through another process if it does not have admin privilege. If the process has admin privilege, it will proceed with the following:</p>
<ol>
<li>Create new service as an auto start to make malware persistent</li>
<li>Change the service description to “Provides support for 3rd party protocol plug-ins for Internet Connection Sharing.”</li>
<li>Start the service.</li>
<li>Collect system information such as process name and system information</li>
<li>Encrypt the collected information via the AES-128 algorithm and SHA1 hash algorithm.</li>
<li>Encrypt the information and POST at the C&amp;C server.</li>
</ol>
<p style="text-align: center;"><a href="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2017/11/171114Emotet_04.jpg"><img src="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2017/11/171114Emotet_04.jpg" alt="Figure 4. EMOTET collects system process information (left) and saves the result to memory (right)"/></a></p>
<p style="text-align: center;"><i>Figure 4. EMOTET collects system </i>process<i> information (left) and saves the result to memory (right)</i></p>
<p style="text-align: center;"><a href="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2017/11/171114Emotet_05.jpg"><img src="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2017/11/171114Emotet_05.jpg"/></a></p>
<p style="text-align: center;"><i>Figure 5. EMOTET collects information about the system version and current applications running under C:\\WOW64\</i></p>
<p style="text-align: center;"><a href="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2017/11/171114Emotet_06.jpg"><img src="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2017/11/171114Emotet_06.jpg"/></a></p>
<p style="text-align: center;"><i>Figure 6. EMOTET C2 IP(</i>RED):PORT<i>(YELLOW) List</i></p>
<p><i><b>Infection Chain</b></i></p>
<p style="text-align: center;"><a href="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2017/11/171114Emotet_07_diagram.jpg"><img src="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2017/11/171114Emotet_07_diagram.jpg"/></a></p>
<p style="text-align: center;"><i>Figure 7. The variant’s infection chain</i></p>
<p>The infection chain of this variant starts with a phishing email. The email contains a malicious URL that will drop a document file containing a malicious macro.</p>
<p style="text-align: center;"><a href="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2017/11/171114Emotet_08.jpg"><img src="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2017/11/171114Emotet_08.jpg" alt="Figure 8. EMOTET phishing email"/></a></p>
<p style="text-align: center;"><i>Figure 8. EMOTET phishing email</i></p>
<p style="text-align: center;"><a href="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2017/11/171114Emotet_09.jpg"><img src="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2017/11/171114Emotet_09.jpg" alt="Figure 9. Malicious EMOTET document"/></a></p>
<p style="text-align: center;"><i>Figure 9. Malicious EMOTET document</i></p>
<p style="text-align: center;"><a href="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2017/11/171114Emotet_10.jpg"><img src="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2017/11/171114Emotet_10.jpg" alt="Figure 10. The malicious macro inside the document will prompt cmd.exe and PowerShell to execute an encoded and obfuscated string."/></a></p>
<p style="text-align: center;"><i>Figure 10. The malicious macro inside the document will prompt cmd.exe and PowerShell to execute an encoded and obfuscated string.</i></p>
<p>The command downloads EMOTET from hxxp<i>://bonn-medien[.]de/RfThRpWC/</i> and will execute the dropper PE payload from the malicious site.</p>
<p style="text-align: center;"><a href="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2017/11/171114Emotet_11.jpg"><img src="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2017/11/171114Emotet_11.jpg" alt="Figure 11. The network traffic of Powershell downloading the dropper"/></a></p>
<p style="text-align: center;"><i>Figure 11. The network traffic of Powershell downloading the dropper from bonn-medien[.]de/RfThRpWC/</i></p>
<p>Enterprises and end-users can avoid threats like EMOTET by following best practices for <a href="https://www.trendmicro.com/vinfo/us/security/news/cybercrime-and-digital-threats/best-practices-identifying-and-mitigating-phishing-attacks" target="_blank">defending against phishing attacks</a>. Users should always be cautious of individuals or organizations that ask for personal information. Most companies will not ask for sensitive data from its customers. When in doubt, users should verify with the company to avoid any potential issues. Users should also avoid clicking links or downloading files even if they come from seemingly “trustworthy” sources. In addition, enterprises can stay protected by employing <a href="https://www.trendmicro.com/vinfo/us/security/news/cybercrime-and-digital-threats/infosec-guide-email-threats" target="_blank">strong security policies</a> to their email gateway and ensuring that their network infrastructure can filter, validate, and block malicious traffic like anomalous data exfiltration.</p>
