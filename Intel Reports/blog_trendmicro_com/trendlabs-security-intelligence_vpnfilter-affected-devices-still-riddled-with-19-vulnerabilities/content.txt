	<h1 class="article-details__title">VPNFilter-affected Devices Still Riddled with 19 Bugs</h1>
	<p class="article-details__description">This blog tackles the VPNFilter malware and if deployed devices are vulnerable to it. Based on our data, plenty of the devices are still using old firmware versions. In fact, 19 known vulnerabilities can still be detected in devices up to this day.</p>
	<p class="article-details__author-by">By: Tony Yang, Peter Lee
		
			<time class="article-details__date">Jul 13, 2018</time>

		<div>
			<p>Our IoT scanning tool allows users to identify if connected devices (e.g. routers, network attached storage devices, IP cameras, and printers) in a given network are vulnerable to security risks and vulnerabilities, such as those related to Mirai, Reaper, and WannaCry.</p>
<p>We gather our data from the <a href="/en_us/forHome/products/homenetworksecurity.html" target="_self">Trend Micro™ Home Network Security</a> solution and <a href="/en_us/forHome/products/housecall/home-network.html" target="_self">HouseCall™ for Home Networks</a> scanner. HouseCall for Home Networks is a free tool that features device recognition and vulnerability scanning in users' networks and connected devices. Home Network Security is a solution plugged into users’ routers that protects connected devices from potential cyberattacks. Our scanning can cover multiple operating systems, including Linux, Mac, Windows, Android, iOS, and other software development kit (SDK) platforms.</p>
<p>This blog tackles the recently ill-famed VPNFilter malware and if deployed devices are vulnerable to it and other vulnerabilities. VPNFilter is a newly discovered, multi-stage malware (detected by Trend Micro as <a href="https://www.trendmicro.com/vinfo/tmr/?/us/threat-encyclopedia/malware/ELF_VPNFILT.A" target="_blank">ELF_VPNFILT.A</a>, <a href="https://www.trendmicro.com/vinfo/tmr/?/us/threat-encyclopedia/malware/ELF_VPNFILT.B" target="_blank">ELF_VPNFILT.B</a>, <a href="https://www.trendmicro.com/vinfo/tmr/?/us/threat-encyclopedia/malware/ELF_VPNFILT.C" target="_blank">ELF_VPNFILT.C</a>, and <a href="https://www.trendmicro.com/vinfo/tmr/?/us/threat-encyclopedia/malware/ELF_VPNFILT.D" target="_blank">ELF_VPNFILT.D</a>) that affects many models of connected devices. Initially <a href="https://www.trendmicro.com/vinfo/tmr/?/us/security/news/internet-of-things/reboot-your-routers-vpnfilter-infected-over-500-000-routers-worldwide" target="_blank">reported</a> at the tail end of May to have infected at least 500,000 networking devices across 54 countries, including those from Linksys, MikroTik, Netgear, and TP-Link, to steal website credentials and even render devices unusable, the malware is now seen <a href="https://www.zdnet.com/article/vpnfilter-malware-now-targeting-asus-d-link-huawei-zte/" target="_blank">targeting more devices</a> to deliver exploits and even override reboots. The Federal Bureau of Investigation (FBI) has even released a <a href="https://www.ic3.gov/media/2018/180525.aspx" target="_blank">public service announcement</a> (PSA), warning that it is the work of foreign threat actors looking to compromise networked devices worldwide.</p>
<h3><b><span class="body-subhead-title">Different brands and models affected by VPNFilter and more</span></b></h3>

<p style="text-align: center;"><i>Table 2. 19 vulnerability detections on VPNFilter-affected devices</i></p>
<p>As expected, the 19 vulnerabilities primarily affect routers. Interestingly, the Authentication Bypass Vulnerability CVE-2015-7261, an FTP (File Transfer Protocol) flaw in the QNAP NAS firmware, mostly affects printers based on our detection. While determining the possible reason behind this, we found that many of the detected printers’ FTP could connect to the network without any authentication. In some cases, this may be the printer’s default configuration, but it still poses a potential security risk if the FTP is set as open on the internet.</p>
<p style="text-align: center;"><a href="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2018/07/shodan-ftp-printer.jpg"><img src="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2018/07/shodan-ftp-printer.jpg" alt="Figure 1. A Shodan result of an FTP connection to a printer without authentication"/></a></p>
<p style="text-align: center;"><i>Figure 1. A Shodan result of an FTP connection to a printer without authentication</i></p>
<p style="text-align: left;">The other vulnerabilities detected, such as the Buffer Overflow <a href="https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2013-0229" target="_blank">CVE-2013-0229</a> and Stack Overflow <a href="https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2013-0230" target="_blank">CVE-2013-0230</a>, can allow attackers to cause a denial-of-service (DoS) and execute arbitrary code in systems, respectively. Vulnerable UPnP Services detected, moreover, aren’t exclusively associated with Netgear/TP-Link/D-Link devices, as other brands could also have the same vulnerability. In that case, we can expect more detections.</p>
<h3 style="text-align: left;"><b><span class="body-subhead-title">Protecting devices and networks against VPNFilter malware and other vulnerabilities</span></b></h3>
<p>The threat of VPNFilter malware is augmented by the fact that other publicly known vulnerabilities were detected in the affected devices. Since not all device manufacturers provide immediate fixes for discovered vulnerabilities and not all users regularly apply patches, users should first secure the way they set up their devices and networks. <a href="/en_us/forHome/products/homenetworksecurity.html" target="_self">Trend Micro™ Home Network Security</a> solution can check internet traffic between the router and all connected devices. Our IoT scanning tool has been integrated into the Home Network Security solution and <a href="/en_us/forHome/products/housecall/home-network.html" target="_self">HouseCall™ for Home Networks</a> scanner. Enterprises can also monitor all ports and network protocols for advanced threats and thwart targeted attacks with the <a href="/en_us/business/products/network/advanced-threat-protection/inspector.html" target="_self">Trend Micro™ Deep Discovery™ Inspector</a> network appliance.</p>
<p>Aside from adopting security solutions that can protect networks and connected devices from the vulnerabilities through the identification and assessment of potential risks, we recommend standard security measures, such as:</p>
<ul>
<li><span class="rte-red-bullet">Updating the firmware versions of devices once they’re available to avoid attacks that exploit known vulnerabilities.</span></li>
<li><span class="rte-red-bullet">Avoiding the use of public Wi-Fi on devices that are also used in home or corporate networks.</span></li>
<li><span class="rte-red-bullet">Changing device’s default credentials and using strong passwords to deter unauthorized access.</span></li>
<li><span class="rte-red-bullet">Being wary of suspicious URLs or attachments from unknown sources that may lead to infecting devices connected to the network.</span></li>
</ul>
<p>Users of the <a href="/en_us/forHome/products/homenetworksecurity.html" target="_self">Trend Micro Home Network Security</a> solution are also protected from particular vulnerabilities via these rules:</p>
<ul>
<li><span class="rte-red-bullet">1058981 WEB Directory Traversal -21</span></li>
<li><span class="rte-red-bullet">1130327 EXPLOIT ASUSWRT 3.0.0.4.376_1071 LAN Backdoor Command Execution (CVE-2014-9583)</span></li>
</ul>
