<div class="cmp-text" id="text-123fb8766a" data-cmp-name="cmp-text-" data-cmp-is="text" data-target-location="false">
    
       <span><p>The June 2018 adversary spotlight is on <b>MUSTANG PANDA, a China-based adversary that has demonstrated an ability to rapidly assimilate new tools and tactics into its operations</b>, as evidenced by its use of exploit code for CVE-2017-0199 within days of its public disclosure. In April 2017, CrowdStrike® Falcon Intelligence™ observed a previously unattributed actor group with a Chinese nexus targeting a U.S.-based think tank. Further analysis revealed a wider campaign with unique tactics, techniques, and procedures (TTPs). This <b>adversary targets non-governmental organizations (NGOs) in general</b>, but uses Mongolian language decoys and themes, suggesting this actor has a specific focus on gathering intelligence on Mongolia. These campaigns involve the <b>use of shared malware like Poison Ivy or PlugX</b>. Recently, Falcon Intelligence observed new activity from MUSTANG PANDA, <b>using a unique infection chain to target likely Mongolia-based victims</b>. This newly observed activity uses a series of redirections and <a href="/en-us/blog/key-characteristics-of-modern-fileless-attacks/">fileless</a>, malicious implementations of legitimate tools to gain access to the targeted systems. Additionally, MUSTANG PANDA actors reused previously-observed legitimate domains to host files.</p>
<h2>Mustang Panda&#39;s Methods</h2>
<p>Mustang Panda&#39;s unique infection chain often takes the following steps:</p>
<ol><li><b>The infection chain used in this attack begins with a weaponized link to a Google Drive folder</b>, obfuscated using the goo.gl link shortening service.</li><li><b>When contacted, the Google Drive link retrieves a zip file, which contains a .lnk file</b> obfuscated as a .pdf file using the double extension trick.</li><li>This file <b>requires the target to attempt to open the .lnk file, which redirects the user to a Windows Scripting Component (.wsc) file</b>, hosted on an adversary-controlled microblogging page. MUSTANG PANDA has previously used the observed microblogging site to host malicious PowerShell scripts and Microsoft Office documents in targeted attacks on Mongolia-focused NGOs.</li><li><b>The .lnk file uses an embedded VBScript component to retrieve a decoy PDF file and a PowerShell script</b> from the adversary-controlled web page.</li><li><b>The PowerShell script creates a Cobalt Strike stager payload.</b> This PowerShell script also retrieves an XOR-encoded Cobalt Strike beacon payload from an adversary-controlled domain.</li><li><b>The Cobalt Strike Beacon implant beacons to the command-and-control (C2) IP address, which is used to remotely control the implant.</b></li></ol>
<p>There are no known community or industry names associated with this actor.</p>
<h2>Other Known China-based Adversaries</h2>
