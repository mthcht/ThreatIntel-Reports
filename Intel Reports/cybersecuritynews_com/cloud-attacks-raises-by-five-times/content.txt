                        <header class="td-post-title">
                            <ul class="td-category"><li class="entry-category"><a  href="https://cybersecuritynews.com/category/cyber-security-news/">Cyber Security News</a></li><li class="entry-category"><a  href="https://cybersecuritynews.com/category/threats/">Threats</a></li></ul>                            <h1 class="entry-title">Cloud Attacks Raises by Five Times Attacking Sensitive IAM Service Accounts</h1>

                            

                            <div class="td-module-meta-info">
                                <div class="td-post-author-name"><div class="td-author-by">By</div> <a href="https://cybersecuritynews.com/author/guru/">Guru Baran</a><div class="td-author-line"> - </div> </div>                                <span class="td-post-date"><time class="entry-date updated td-module-date" datetime="2025-03-27T15:29:41+00:00" >March 27, 2025</time></span>                                                                                            </div>

                        </header>

                                            </div>
                </div>
            </div> <!-- /.td-pb-row -->

            <div class="td-pb-row">
                                            <div class="td-pb-span8 td-main-content" role="main">
                                <div class="td-ss-main-content">
                                    



        <div class="td-post-content tagdiv-type">
            <div class="td-post-featured-image"><a href="https://i3.wp.com/blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEjp8TUz4i47h_-fuLxoA5egoCZ5rZ1PpPHe4TXunopoCVZDZtUPwBmqtP1S8azrFPzd8gAfFyLoVJyJSc0QXjktv0aNQddXcvzyo5deK_4xGKiHbfZc6ev27oUXwy_g-YpZ9JzClxgC97U7WrhbashyhAFTK113xal_S7PVadDC9_yLMJABxkVACvUNlod1/s16000/Cloud%20Attacks%20Raises%20By%20Five%20Times.webp?w=1600&resize=1600,900&ssl=1" data-caption=""><img post-id="97502" fifu-featured="1" width="696" height="" class="entry-thumb td-modal-image" src="https://i3.wp.com/blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEjp8TUz4i47h_-fuLxoA5egoCZ5rZ1PpPHe4TXunopoCVZDZtUPwBmqtP1S8azrFPzd8gAfFyLoVJyJSc0QXjktv0aNQddXcvzyo5deK_4xGKiHbfZc6ev27oUXwy_g-YpZ9JzClxgC97U7WrhbashyhAFTK113xal_S7PVadDC9_yLMJABxkVACvUNlod1/s16000/Cloud%20Attacks%20Raises%20By%20Five%20Times.webp?w=696&resize=696,0&ssl=1" alt="Cloud Attacks Raises by Five Times Attacking Sensitive IAM Service Accounts" title="Cloud Attacks Raises by Five Times Attacking Sensitive IAM Service Accounts" title=""/></a></div>
            
<p>Organizations are facing an unbelievable surge in cloud-based security threats, with attacks nearly five times more frequent at the end of 2024 compared to the beginning of the year.</p>



<p>Most concerning is the targeted attack on <a href="https://cybersecuritynews.com/identity-management-solutions/" target="_blank" rel="noreferrer noopener">Identity and Access Management (IAM)</a> tokens, which security researchers describe as &#8220;holding the keys to the cloud kingdom.&#8221;</p>



<p>The comprehensive analysis by Unit 42 highlights a 388% overall increase in cloud security alerts throughout 2024, with high-severity alerts climbing by an alarming 235%.&nbsp;</p>



<figure class="wp-block-image"><img decoding="async" src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXcWlScpg8YBklHYnx_9-fYUBv3y-lrK7MTsY6Efs_U3Nprt6_nkFtTUwLngl863cvcDT5oz1KeoSU6JLpIvUNCsKoSGC_az2e8h671EdTNegW6fNQhhbHCTTagf6HwesAJHXxHC?key=BxzBVU66BBm4JF2CQNCOtsba" alt=""/></figure>



<p>These figures represent a significant escalation in both the volume and sophistication of attacks against cloud environments. &#8220;Identity is the defense perimeter of cloud infrastructure,&#8221; states the Unit 42 <a href="https://unit42.paloaltonetworks.com/2025-cloud-security-alert-trends/" target="_blank" rel="noreferrer noopener nofollow">report</a>. </p>



<p>&#8220;Attackers target IAM tokens and credentials as they hold the keys to the cloud kingdom, allowing attackers to move laterally, escalate their permissions and perform additional malicious operations.&#8221;</p>



<p class="has-text-align-center has-background" style="background:linear-gradient(180deg,rgb(238,238,238) 98%,rgb(169,184,195) 100%)">Stop attacks before they start, powered by a 97% precise neural Network to <strong><a href="https://firstwatch.whoisxmlapi.com/?utm_campaign=9753186-Cyber%20Security%20News&amp;utm_source=email&amp;utm_medium=paidemail&amp;utm_content=csn2_websitevisit" target="_blank" rel="noreferrer noopener nofollow">Detect Cyber Attacks</a></strong></p>



<h2 class="wp-block-heading"><strong>High-Severity Cloud Alerts</strong></h2>



<p>Among the most troubling trends identified was the surge in remote command-line access events utilizing IAM tokens and credentials, which increased threefold in 2024.&nbsp;</p>



<p>By December, the average cloud environment experienced over 200 alerts for remote command-line usage of serverless function IAM tokens, compared to just two such alerts in January.</p>



<p>The research documented several other concerning patterns:</p>



<ul class="wp-block-list">
<li>116% rise in IAM-based &#8220;impossible travel&#8221; alerts</li>



<li>60% increase in IAM API requests from outside authorized regions</li>



<li>45% increase in cloud snapshot exports</li>



<li>305% jump in suspicious downloads of multiple cloud storage objects</li>
</ul>



<p>Security researchers noted these attacks align with findings from their 2024 State of Cloud-Native Security Report, which found 71% of organizations attributing increased vulnerability exposures to accelerated deployments, while 45% reported a rise in advanced persistent threat (APT) attacks.</p>



<p>A particularly alarming example involves a ransomware campaign that harvested over 90,000 credentials from 110,000 targeted domains, including nearly 1,200 cloud IAM credentials.&nbsp;</p>



<p>These stolen credentials enabled successful extortion attacks against multiple organizations.</p>



<p>The concentration of attacks on serverless functions has drawn particular concern. According to the research, serverless functions are designed to operate autonomously, and remote usage of a serverless function&#8217;s IAM token indicates compromise and potential lateral movement within cloud environments.</p>



<p>Experts recommend implementing Cloud Detection and Response (CDR) tools alongside traditional <a href="https://cybersecuritynews.com/best-cloud-security-tools/" target="_blank" rel="noreferrer noopener">Cloud Security</a> Posture Management (CSPM) solutions. </p>



<p>While CSPM tools focus on configuration vulnerabilities, CDR provides runtime monitoring to detect malicious activities as they occur.</p>



<h2 class="wp-block-heading"><strong>Recommendations for Enhanced Cloud Security</strong></h2>



<p>Key recommendations include:</p>



<ul class="wp-block-list">
<li>Deploying CDR security for all cloud environments.</li>



<li>Ensuring runtime-enabled agents on mission-critical cloud endpoints.</li>



<li>Implementing cloud audit log monitoring from CSP providers.</li>



<li>Restricting CSP regions where compute and serverless functions can operate.</li>



<li>Enforcing least-privilege architecture for <a href="https://cybersecuritynews.com/identity-management-solutions/" target="_blank" rel="noreferrer noopener">IAM credentials</a>.</li>



<li>Enabling cloud storage versioning and encryption.</li>
</ul>



<p>&#8220;Given the increasing threats targeting cloud environments, the only real defense is to require cloud-based agents for publicly exposed and critical cloud endpoints,&#8221; the report concludes, emphasizing that runtime monitoring and response capabilities are essential to prevent malicious operations within cloud infrastructures.</p>
