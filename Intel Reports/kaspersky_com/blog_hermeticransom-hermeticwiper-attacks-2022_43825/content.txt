				<h1 class="c-article__title">Ransomware as a distraction</h1>	<div class="c-article__intro">
					<p>HermeticRansom cryptor was used as a distraction to support HermeticWiper attacks.</p>
			</div>
	<ul class="c-article__meta-list">
		<li class="c-article__author">
			<div class="c-article__author-avatar">
				<a href="https://www.kaspersky.com/blog/author/editorialteam/"><img alt='' src='https://media.kasperskydaily.com/wp-content/uploads/2025/05/23111210/Midori-Kuma-Ed-192x192.jpg' srcset='https://media.kasperskydaily.com/wp-content/uploads/2025/05/23111210/Midori-Kuma-Ed-192x192.jpg 2x' class='avatar avatar-50 photo' height='50' width='50' decoding='async'/></a>
			</div>
			<p class="c-article__author-name">
				<a href="https://www.kaspersky.com/blog/author/editorialteam/">Editorial Team</a>
			</p>
		</li>
				<li class="c-article__time c-article__meta-spacer">
			<p>
				<time datetime="2022-03-01T11:05:52-05:00">March 1, 2022</time>
			</p>
		</li>
			</ul>
		<figure class="c-article__figure">
		<img width="1460" height="960" src="https://media.kasperskydaily.com/wp-content/uploads/sites/92/2022/03/01110327/hermeticransom-hermeticwiper-attacks-2022-featured.jpg" class="attachment-daily-2019-article-image size-daily-2019-article-image wp-post-image" alt="" decoding="async" fetchpriority="high" data-src="https://media.kasperskydaily.com/wp-content/uploads/sites/92/2022/03/01110327/hermeticransom-hermeticwiper-attacks-2022-featured.jpg" data-srcset="" srcset="" />	</figure>

			<div class="o-row c-article__body c-article__body--columns@lg">
				<div class="o-col-8@lg">
					<div class="c-article__content">
						<div class="c-wysiwyg">
							<p>Our researchers analyzed the HermeticRansom malware also known as Elections GoRansom. By and large, this is a fairly simple cryptor. What is interesting in this case is the purpose for which attackers are using it.</p>
<h2>HermeticRansom goals</h2>
<p>HermeticRansom attacked computers at the same time as another malware known as HermeticWiper, and based on publicly available information from security community, it was used in recent cyberattacks in Ukraine. According to our experts, the relative simplicity and questionable malware workflow implementation suggests that HermeticRansom was used as a smokescreen for HermeticWiper attacks.</p>
<h2>What HermeticRansom is capable of</h2>
<p>Once infecting the victim’s computer, the malware first identifies hard drives and collects a list of directories and files located everywhere except for the Windows and Program Files folders. It then encrypts certain categories of files and renames them adding а .encrypted tag and the email address of the ransomware operators. The malware also creates a read_me.html file in the Desktop folder containing a ransom note with the attackers’ contacts. The note looks like this:</p>
<div id="attachment_43826" style="width: 983px" class="wp-caption aligncenter"><a href="https://media.kasperskydaily.com/wp-content/uploads/sites/92/2022/03/01110106/hermeticransom-hermeticwiper-attacks-2022-featured-ransom-note.png"><img decoding="async" aria-describedby="caption-attachment-43826" src="https://media.kasperskydaily.com/wp-content/uploads/sites/92/2022/03/01110106/hermeticransom-hermeticwiper-attacks-2022-featured-ransom-note.png" alt="Ransom note left by HermeticRansom malware" width="973" height="469" class="size-full wp-image-43826"></a><p id="caption-attachment-43826" class="wp-caption-text">Ransom note left by HermeticRansom malware</p></div>
<p>HermeticRansom encrypts files with following extensions: .inf, .acl, .avi, .bat, .bmp, .cab, .cfg, .chm, .cmd, .com, .crt, .css, .dat, .dip, .dll, .doc, .dot, .exe, .gif, .htm, .ico, .iso, .jpg, .mp3, .msi and odt.</p>
<h2>HermeticRansom peculiarities</h2>
<p>HermeticRansom is written in Golang. It does not use any obfuscation mechanisms, and the encryption method itself is rather cumbersome and inefficient. Judging by these and some other signs, our experts think that this malware was created in a hurry.</p>
<p>You can find a more detailed technical analysis of the malware along with indicators of compromise <a href="https://securelist.com/elections-goransom-and-hermeticwiper-attack/105960/" target="_blank" rel="noopener">on our Securelist blog.</a></p>
<h2>How to stay safe</h2>
<p>Kaspersky Lab security solutions successfully detect HermeticRansom malware and similar threats. We have a range of tools to protect both home computers and corporate infrastructure, including:</p>

</li><li><a href="https://www.kaspersky.com/small-to-medium-business-security/cloud?icid=gl_kdailyplacehold_acq_ona_smm__onl_b2b_kasperskydaily_wpplaceholder____kescloud___" target="_blank" rel="noopener nofollow">Kaspersky Endpoint Security Cloud</a>: our solution for business protection;</li>
