          }

<h2 style="font-weight: bold;"><span style="font-size: 20px; color: #000000;">BlackByte Ransomware Group</span></h2>
<p style="text-align: justify;"><span style="font-size: 17px; color: #000000;">The</span><span style="font-size: 17px; color: #d60059;"> BlackByte </span><span style="font-size: 17px; color: #000000;">group is a Ransomware-as-a-Service (RaaS) operator and started its ransomware operation in July 2021. Since then, they have targeted US-based organizations in critical infrastructure sectors such as government, finance, and food &amp; agriculture. Also, they compromised the network of the famous American football team, San Francisco 49ers, and released some of the confidential data belonging to the team as proof of the attack.&nbsp;</span></p>

<h2 style="font-weight: bold;"><span style="font-size: 20px; color: #000000;">What is BlackByte Ransomware?</span></h2>
<p style="text-align: justify;"><span style="font-size: 17px; color: #d60059;">BlackByte</span><span style="font-size: 17px; color: #0e101a;"> ransomware is the collective name of the ransomware variants from the BlackByte RaaS group. The ransomware was first reported back in July 2021. It exploits ProxyShell vulnerabilities found in Microsoft Exchange Server for initial access. The patch for these vulnerabilities is available. However, unpatched systems are falling victim to these ransomware attacks. Check out </span><a href="/resource/simulating-and-preventing-proxyshell-exchange-exploits" style="font-size: 17px;" rel="noopener" target="_blank"><span>our blog post</span></a><span style="font-size: 17px; color: #0e101a;"> and learn how to prevent the exploitation of ProxyShell vulnerabilities.</span></p>
<p style="text-align: justify;"><span style="font-size: 17px; color: #0e101a;">The ransomware does not attack the infected systems if the language setting is Russian or the languages of former Soviet republics. This behavior is similar to some other ransomware threat groups, as explained in our previous ransomware blog, </span><a href="/resource/lockbit-2.0-ransomware-ttps-used-in-emerging-ransomware-campaigns" style="font-size: 17px; color: #4a6ee0;" rel="noopener" target="_blank"><span>LockBit 2.0</span></a><span style="font-size: 17px; color: #0e101a;">.&nbsp;</span></p>
<p style="text-align: justify;"><span style="font-size: 17px; color: #d60059;">BlackByte</span><span style="font-size: 17px; color: #0e101a;"> ransomware variants only use symmetric encryption. In their earlier ransomware variants, The </span><span style="font-size: 17px; color: #d60059;">BlackByte threat group</span><span style="font-size: 17px; color: #0e101a;"> distributed the encryption key to every victim from their command and control (C2) server in a </span><span style="font-size: 17px; color: #d60059;">.png</span><span style="font-size: 17px; color: #0e101a;"> file. Since the same encryption key is used for every victim, Trustwave was able to devise a </span><span style="font-size: 17px; color: #d60059;">global decryptor</span><span style="font-size: 17px; color: #0e101a;"> [2]. After the release of the global decryptor, the ransomware group stopped delivering the encryption key from the C2 server and changed the key. Although the decryptor might not work in some cases, it is worth a try as it does not harm already encrypted files.&nbsp;</span></p>
<p style="text-align: justify;"><span style="font-size: 17px; color: #0e101a;">After encrypting the victim files, BlackByte ransomware appends the</span><span style="font-size: 17px; color: #d60059;"> .blackbyte</span><span style="font-size: 17px; color: #0e101a;"> extension. The ransomware leaves the same ransom note in all encrypted directories, and the ransom note includes a</span><span style="font-size: 17px; color: #d60059;"> .onion</span><span style="font-size: 17px; color: #0e101a;"> link that instructs the victim how to pay the ransom and receive the decryption key. Also, the ransom note claims that the ransomware has exfiltrated data from its victims to scare its victims to pay the ransom.</span></p>

<h3 style="font-weight: bold;"><span style="font-size: 17px; color: #000000;">Credential Access</span></h3>
<ul>
<li style="color: #000000;" aria-level="1"><strong><span style="font-size: 17px; color: #000000;">T1003 OS Credential Dumping</span></strong></li>
</ul>
<p style="padding-left: 40px;"><span style="font-size: 17px; color: #d60059;">BlackByte</span><span style="font-size: 17px; color: #000000;"> group uses </span><span style="font-size: 17px; color: #d60059;">Cobalt Strike</span><span style="font-size: 17px; color: #000000;"> to dump credentials and access service accounts in the victim network.</span><span></span></p>
<h3 style="font-weight: bold;"><span style="font-size: 17px; color: #000000;">Discovery</span></h3>

<h3 style="font-weight: bold;"><span style="font-size: 17px; color: #000000;">Lateral Movement</span></h3>
<ul>
<li style="color: #000000;" aria-level="1"><strong><span style="font-size: 17px; color: #000000;">T1021.002 Remote Services: SMB/Windows Admin Shares</span></strong></li>
</ul>
<p style="padding-left: 40px;"><span style="font-size: 17px; color: #d60059;">BlackByte</span><span style="font-size: 17px; color: #000000;"> ransomware creates </span><span style="font-size: 17px; color: #d60059;">SMB</span> <span style="font-size: 17px; color: #d60059;">shares</span><span style="font-size: 17px; color: #000000;"> to distribute </span><span style="font-size: 17px; color: #d60059;">AnyDesk</span><span style="font-size: 17px; color: #000000;">, a remote desktop application, to other assets in the victim’s network using </span><span style="font-size: 17px; color: #d60059;">Cobalt</span> <span style="font-size: 17px; color: #d60059;">Strike</span><span style="font-size: 17px; color: #000000;">.</span><span></span></p>
<h3 style="font-weight: bold;"><span style="font-size: 17px; color: #000000;">Collection</span></h3>
<ul>
<li style="color: #000000;" aria-level="1"><strong><span style="font-size: 17px; color: #000000;">T1560.001 Archive Collected Data: Archive via Utility</span></strong></li>
</ul>
<p style="padding-left: 40px;"><span style="font-size: 17px; color: #d60059;">BlackByte</span><span style="font-size: 17px; color: #000000;"> compresses the victim's file before exfiltration.</span></p>
<h3 style="font-weight: bold;"><span style="font-size: 17px; color: #000000;">Command and Control (C2)</span></h3>
<ul>
<li style="color: #000000;" aria-level="1"><strong><span style="font-size: 17px; color: #000000;">T1105 Ingress Tool Transfer</span></strong></li>
</ul>
<p style="padding-left: 40px;"><span style="font-size: 17px; color: #000000;">The </span><span style="font-size: 17px; color: #d60059;">BlackByte</span><span style="font-size: 17px; color: #000000;"> group transfers a </span><span style="font-size: 17px; color: #d60059;">Cobalt Strike beacon</span><span style="font-size: 17px; color: #000000;"> to the victim using the webshell they placed. After the beacon is placed, they transfer the </span><span style="font-size: 17px; color: #d60059;">AnyDesk</span><span style="font-size: 17px; color: #000000;"> application.</span><span></span></p>
<h3 style="font-weight: bold;"><span style="font-size: 17px; color: #000000;">Exfiltration</span></h3>
<ul>
<li style="color: #000000;" aria-level="1"><strong><span style="font-size: 17px; color: #000000;">T1567.002 Exfiltration Over Web Service: Exfiltration to Cloud Storage</span></strong></li>
</ul>
<p style="padding-left: 40px;"><span style="font-size: 17px; color: #d60059;">BlackByte</span><span style="font-size: 17px; color: #000000;"> ransomware sends victim’s compressed files to anonymous file-sharing services such as </span><span style="font-size: 17px; color: #d60059;">anonymfiles.com</span><span style="font-size: 17px; color: #000000;"> and</span><span style="font-size: 17px; color: #d60059;"> file.io</span><span style="font-size: 17px; color: #000000;">.</span><span></span></p>
<h3 style="font-weight: bold;"><span style="font-size: 17px; color: #000000;">Impact</span></h3>
<ul>
<li style="color: #000000;" aria-level="1"><strong><span style="font-size: 17px; color: #000000;">T1486 Data Encrypted for Impact</span></strong></li>
</ul>
<p style="padding-left: 40px;"><span style="font-size: 17px; color: #d60059;">BlackByte</span><span style="font-size: 17px; color: #000000;"> uses symmetric key encryption to encrypt the victim’s files. Check out </span><a href="https://www.picussecurity.com/resource/the-most-common-ransomware-ttp-mitre-attck-t1486-data-encrypted-for-impact" style="font-size: 17px;"><span>our blog post</span></a><span style="font-size: 17px; color: #000000;"> to learn more detail on this MITRE ATT&amp;CK technique.</span></p>
