							<h2>CaddyWiper, a new data wiper hits Ukraine</h2>
						</div>
						<div class="post-time mb-3">
							<span><i><img src="https://securityaffairs.com/wp-content/themes/security_affairs/images/user-icon.svg" alt="" /></i> <a href="https://securityaffairs.com/author/paganinip">Pierluigi Paganini</a></span>
							<span><i><img src="https://securityaffairs.com/wp-content/themes/security_affairs/images/clock-icon.svg" alt="" /></i> March 15, 2022</span>
						</div>
						<img class="img-fluid mb-4" src="https://i0.wp.com/securityaffairs.com/wp-content/uploads/2022/03/CaddyWiper.png?fit=1500%2C888&ssl=1" alt="" />
						<div style="margin-top: 0px; margin-bottom: 0px;" class="sharethis-inline-share-buttons" ></div>
<h2 class="wp-block-heading">Experts discovered a new wiper, tracked as CaddyWiper, that was employed in attacks targeting Ukrainian organizations.<gwmw style="display:none;"><gwmw style="display:none;"></h2>



<p>Experts at ESET Research Labs discovered a new data wiper, dubbed CaddyWiper, that was employed in attacks targeting Ukrainian organizations.</p>



<p>The security firm has announced the discovery of the malware with a series of tweets:</p>



<figure class="wp-block-embed is-type-rich is-provider-twitter wp-block-embed-twitter"><div class="wp-block-embed__wrapper">
<blockquote class="twitter-tweet" data-width="500" data-dnt="true"><p lang="en" dir="ltr"><a href="https://twitter.com/hashtag/BREAKING?src=hash&amp;ref_src=twsrc%5Etfw">#BREAKING</a> <a href="https://twitter.com/hashtag/ESETresearch?src=hash&amp;ref_src=twsrc%5Etfw">#ESETresearch</a> warns about the discovery of a 3rd destructive wiper deployed in Ukraine 🇺🇦. We first observed this new malware we call <a href="https://twitter.com/hashtag/CaddyWiper?src=hash&amp;ref_src=twsrc%5Etfw">#CaddyWiper</a> today around 9h38 UTC. 1/7 <a href="https://t.co/gVzzlT6AzN">pic.twitter.com/gVzzlT6AzN</a></p>&mdash; ESET Research (@ESETresearch) <a href="https://twitter.com/ESETresearch/status/1503436420886712321?ref_src=twsrc%5Etfw">March 14, 2022</a></blockquote><script async src="https://platform.twitter.com/widgets.js" charset="utf-8"></script>
</div></figure>



<figure class="wp-block-embed is-type-rich is-provider-twitter wp-block-embed-twitter"><div class="wp-block-embed__wrapper">
<blockquote class="twitter-tweet" data-width="500" data-dnt="true"><p lang="en" dir="ltr">CaddyWiper does not share any significant code similarity with <a href="https://twitter.com/hashtag/HermeticWiper?src=hash&amp;ref_src=twsrc%5Etfw">#HermeticWiper</a>, <a href="https://twitter.com/hashtag/IsaacWiper?src=hash&amp;ref_src=twsrc%5Etfw">#IsaacWiper</a> or any other malware known to us. The sample we analyzed was not digitally signed. 3/7<a href="https://t.co/EGp9NnctD9">https://t.co/EGp9NnctD9</a></p>&mdash; ESET Research (@ESETresearch) <a href="https://twitter.com/ESETresearch/status/1503436425676705807?ref_src=twsrc%5Etfw">March 14, 2022</a></blockquote><script async src="https://platform.twitter.com/widgets.js" charset="utf-8"></script>
</div><figcaption><gwmw style="display:none;"></figcaption></figure>



<p><em>&#8220;This new malware erases user data and partition information from attached drives,&#8221; ESET Research Labs&nbsp;<a href="https://twitter.com/ESETresearch/status/1503436423818534915" target="_blank" rel="noreferrer noopener">reported</a>. &#8220;ESET telemetry shows that it was seen on a few dozen systems in a limited number of organizations.&#8221;</em></p>



<p>CaddyWiper is the third wiper observed by ESET in attacks against Ukraine after <a href="https://securityaffairs.co/wordpress/128349/malware/wiper-malware-hermeticwipe-ukrain.html">HermeticWiper</a> and <a href="https://securityaffairs.co/wordpress/128553/malware/isaacwiper-data-wiper.html">IsaacWiper</a>, experts pointed out that it does not share any significant code similarity with them.</p>



<p>Similar to HermeticWiper deployments, CaddyWiper being deployed via GPO, a circumstance that suggests the attackers had initially compromised the target&#8217;s Active Directory server.</p>



<p>In order to maintain access to the target organization while still disturbing operations, the CaddyWiper avoids destroying data on domain controllers. CaddyWiper uses the DsRoleGetPrimaryDomainInformation() function to determine if a device is a domain controller.</p>



<figure class="wp-block-embed is-type-rich is-provider-twitter wp-block-embed-twitter"><div class="wp-block-embed__wrapper">
<blockquote class="twitter-tweet" data-width="500" data-dnt="true"><p lang="en" dir="ltr">Information from the PE header of CaddyWiper suggests it was compiled the same day it was deployed to targeted networks. 6/7 <a href="https://t.co/Ay363IRGzX">pic.twitter.com/Ay363IRGzX</a></p>&mdash; ESET Research (@ESETresearch) <a href="https://twitter.com/ESETresearch/status/1503436437848571914?ref_src=twsrc%5Etfw">March 14, 2022</a></blockquote><script async src="https://platform.twitter.com/widgets.js" charset="utf-8"></script>
</div><figcaption><gwmw style="display:none;"></figcaption></figure>



<p>The CaddyWiper sample analyzed by ESET was not digitally signed, the malware was compiled.</p>



<p>Microsoft researchers also observed another wiper that was employed in attacks against Ukraine, it was tracked as <a href="https://securityaffairs.co/wordpress/128534/malware/cisa-and-fbi-warn-of-potential-data-wiping-attacks-spillover.html">WhisperGate</a>.</p>



<p>In Mid-February, the Security Service of Ukraine (SSU) today revealed the country was the target of an ongoing “wave of hybrid warfare” conducted by Russia-linked malicious actors. Threat actors aim at destabilizing the social contest in the country and instilling fear and untrust in the country’s government. Data wiper usage was part of this <a href="https://securityaffairs.co/wordpress/128019/cyber-warfare-2/russia-targets-ukraine-hybrid-warfare.html">hybrid warfare strategy.</a></p>
