	function supportsWorkerOffloading() {

	function getSessionSupportTests() {

				typeof item === 'object' &&
				typeof item.timestamp === 'number' &&

	function emojiSetsRenderIdentically( context, set1, set2 ) {

		context.fillText( set1, 0, 0 );

				0,

				 * Test for Transgender flag compatibility. Added in Unicode 13.
				 *
				 * To test for support, we try to render it, and compare the rendering to how it would look if

				);

					return false;
				}

				 * Test for UN flag compatibility. This is the least supported of the letter locale flags,
				 * so gives us an easy test for full support.
				 *
				 * To test for support, we try to render it, and compare the rendering to how it would look if
				 * the browser doesn't render it correctly ([U] + [N]).
				 */

					context,
					'\uD83C\uDDFA\uD83C\uDDF3', // as the sequence of two code points
					'\uD83C\uDDFA\u200B\uD83C\uDDF3' // as the two code points separated by a zero-width space
				);

				if ( isIdentical ) {
					return false;
				}

				 * Test for English flag compatibility. England is a country in the United Kingdom, it
				 * does not have a two letter locale code but rather a five letter sub-division code.
				 *
				 * To test for support, we try to render it, and compare the rendering to how it would look if
				 * the browser doesn't render it correctly (black flag emoji + [G] + [B] + [E] + [N] + [G]).
				 */
				isIdentical = emojiSetsRenderIdentically(
					context,
					// as the flag sequence
					'\uD83C\uDFF4\uDB40\uDC67\uDB40\uDC62\uDB40\uDC65\uDB40\uDC6E\uDB40\uDC67\uDB40\uDC7F',
					// with each code point separated by a zero-width space
					'\uD83C\uDFF4\u200B\uDB40\uDC67\u200B\uDB40\uDC62\u200B\uDB40\uDC65\u200B\uDB40\uDC6E\u200B\uDB40\uDC67\u200B\uDB40\uDC7F'
				);

				return ! isIdentical;
			case 'emoji':
				/*
				 * Rise Like a Phoenix.
				 *
				 * To test for Emoji 15.1 support, try to render a new emoji: Phoenix.
				 *
				 * A phoenix, a mythical immortal bird with flame-like feathers found in the folklore of many global
				 * cultures. Often used to symbolize renewal or rebirth.
				 *
				 * The Phoenix emoji is a ZWJ sequence combining 🐦 Bird, Zero Width Joiner and 🔥 Fire.
				 * These display as a single emoji on supported platforms.
				 *

				);

				return ! isIdentical;

			typeof WorkerGlobalScope !== 'undefined' &&

		 * check for bold rendering support to avoid invisible emoji in Chrome.

			supports[ test ] = browserSupportsEmoji( context, test, emojiSetsRenderIdentically );

				settings.DOMReady = true;
			};

			return domReadyPromise;

					addScript( src.concatemoji );
				} else if ( src.wpemoji && src.twemoji ) {
					addScript( src.twemoji );
					addScript( src.wpemoji );
				}
			}
		} );

							<h2>Ali Baba, the APT group from the Middle East</h2>
						</div>
						<div class="post-time mb-3">
							<span><i><img src="https://securityaffairs.com/wp-content/themes/security_affairs/images/user-icon.svg" alt="" /></i> <a href="https://securityaffairs.com/author/paganinip">Pierluigi Paganini</a></span>
							<span><i><img src="https://securityaffairs.com/wp-content/themes/security_affairs/images/clock-icon.svg" alt="" /></i> February 17, 2015</span>
						</div>
						<img class="img-fluid mb-4" src="https://i0.wp.com/securityaffairs.com/wp-content/uploads/2015/02/Ali-Baba-APT.png?fit=518%2C310&ssl=1" alt="" />
						<div style="margin-top: 0px; margin-bottom: 0px;" class="sharethis-inline-share-buttons" ></div><h2>Adrian Nish of BAE System presented the results of its investigation on the Ali Baba APT group operating from the Middle East that hit Western companies.</h2>
<p>Yesterday the Kaspersky Lab team revealed the results of its investigation on the hacking crew dubbed <a title="The Equation Group shows most complex and sophisticated hacking techniques ever seen" href="http://securityaffairs.co/wordpress/33637/cyber-crime/the-equation-group-atp.html" target="_blank">the Equation group</a>, a team of hackers that demonstrate extraordinary capabilities and sophisticated tactics, techniques, and procedures. Unfortunately, the number of <a title="Pitty Tiger – small ATPs scare private companies" href="http://securityaffairs.co/wordpress/26592/cyber-crime/pitty-tiger-atp.html" target="_blank">ATPs</a> is growing over the years, the majority of them goes under the radar for a long period.</p>
<p>In 2013, Adrian Nish of BAE Systems investigated on a cyber attack suffered by an engineering company in the UK that operates in the national power industry. The security experts discovered that hackers have compromised the company network for some time, exfiltrating any kind of information.</p>
<blockquote><p><em>“The group has probably been working for about two years now,” Nish explained. “It’s an emerging trend in the Middle East. That’s a complicated region and the offensive side of things is becoming complicated there too. There’s offensive cyber companies and local malware authoring now.”</em></p></blockquote>
<p>Nish identified the C&amp;C servers used by the threat actors and discovered that Google was indexing some of the machines used by the hackers to siphon data. According to the researcher, the bad actors could be members of a pro-Iranian group and proved to have access to a wide set of hacking tools.</p>
<p>BAE firm dubbed the APT group Ali Baba because a code name in one of the tools belonging to their arsenal.</p>
<blockquote><p><em>“They had taken network diagrams, usernames and credentials from an Israeli university and even an entire Web app that they stole from a group in the Middle East,” Nish said in a talk at the Kaspersky Lab <a href="http://sas.kaspersky.com/">Security Analyst Summit</a> here Monday. “They had even stolen some signatures, physical signatures from people who had scanned them for some reason. What could possibly go wrong with that?”</em></p></blockquote>
<p>Nish confirmed to have discovered nearly 40 distinct hacking tools, including five modules of custom malware, a key logger, a custom hash cracker and many others. The expert highlighted some interesting methods for defeating incident response on compromised networks and for data exfiltration.</p>
<p>Nish detailed one of the tools in the arsenal of the Ali Baba APT, Fakeddos.exe, that was used the hackers to generate large amounts of junk traffic on compromised networks, a tactic used by the threat actor to overwrite the logs of legitimate traffic making difficult investigation from security firms.</p>
<blockquote><p><em>“That really makes incident response quite a pain, really,” Nish said.</em></p></blockquote>
<p>Ali Baba hackers used a singular <a title="Hackers used data exfiltration based on video steganography" href="http://securityaffairs.co/wordpress/30624/cyber-crime/hackers-used-data-exfiltration-based-video-steganography.html" target="_blank">exfiltration</a> technique based on email, they disguised the outbound emails as Viagra spam messages to avoid detection of defense systems.</p>
<p>According to a <a href="http://www.cylance.com/assets/Cleaver/Cylance_Operation_Cleaver_Report.pdf">report</a> published by the security company Cylance, the UK firm wasn&#8217;t the unique known victim of the Ali Baba, the APT also had compromised transportation companies in South Korea and Pakistan. Cylance identified the hacking team as <a title="Operation Cleaver – Iranian hackers target industries worldwide" href="http://securityaffairs.co/wordpress/30734/intelligence/operation-cleaver-iranian-hackers.html" target="_blank">OpCleaver</a>.</p>
<p><a href="https://i0.wp.com/securityaffairs.co/wordpress/wp-content/uploads/2015/02/Ali-Baba-geo.png"><img data-recalc-dims="1" fetchpriority="high" decoding="async" class="aligncenter wp-image-33707" src="https://i0.wp.com/securityaffairs.co/wordpress/wp-content/uploads/2015/02/Ali-Baba-geo.png?resize=480%2C309" alt="Ali Baba geo" width="480" height="309" srcset="https://i0.wp.com/securityaffairs.com/wp-content/uploads/2015/02/Ali-Baba-geo.png?w=693&amp;ssl=1 693w, https://i0.wp.com/securityaffairs.com/wp-content/uploads/2015/02/Ali-Baba-geo.png?resize=300%2C193&amp;ssl=1 300w" sizes="(max-width: 480px) 100vw, 480px" /></a></p>
