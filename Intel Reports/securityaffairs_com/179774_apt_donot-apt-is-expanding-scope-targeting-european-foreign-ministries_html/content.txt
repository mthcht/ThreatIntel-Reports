	function supportsWorkerOffloading() {

							<h2>DoNot APT is expanding scope targeting European foreign ministries</h2>
						</div>
						<div class="post-time mb-3">
							<span><i><img src="https://securityaffairs.com/wp-content/themes/security_affairs/images/user-icon.svg" alt="" /></i> <a href="https://securityaffairs.com/author/paganinip">Pierluigi Paganini</a></span>
							<span><i><img src="https://securityaffairs.com/wp-content/themes/security_affairs/images/clock-icon.svg" alt="" /></i> July 10, 2025</span>
						</div>
						<img class="img-fluid mb-4" src="https://i0.wp.com/securityaffairs.com/wp-content/uploads/2025/07/image-11.png?fit=1335%2C820&ssl=1" alt="" />
						<div style="margin-top: 0px; margin-bottom: 0px;" class="sharethis-inline-share-buttons" ></div>
<h2 class="wp-block-heading">DoNot APT, likely an India-linked cyberespionage group, targets European foreign ministries with LoptikMod malware.</h2>



<p>The <a href="https://securityaffairs.com/173257/apt/donot-team-android-malware.html">DoNot APT</a> group, likely linked to India, has expanded its operations and is targeting European foreign ministries with a new malware, called LoptikMod.</p>



<p>The Donot Team (<span style="box-sizing: border-box; margin: 0px; padding: 0px;">also known as <a href="https://securityaffairs.com/134674/apt/donot-team-improves-jaca-framework.html" target="_blank">APT-C-35</a> and Origami Elephant) has been active since</span> 2016, focusing on government entities, foreign ministries, defense organizations, and NGOs in South Asia and Europe.</p>



<p>DoNot APT uses custom Windows malware via phishing for espionage, enabling long-term access and data theft. In a recent campaign, analyzed by cybersecurity firm Trellix, the cyber spies used LoptikMod malware to steal sensitive data from infected systems.</p>



<p>Attackers used a spear-phishing email impersonating defense officials to target a European diplomatic entity, delivering the LoptikMod malware via a password-protected RAR file. </p>



<p>The phishing email used HTML and UTF-8 to appear legitimate, including proper special characters. It linked to a password-protected RAR file on Google Drive. The archive contained a disguised executable (notflog.exe) with a PDF icon to trick users into running malware.<gwmw style="display:none;"></gwmw></p>



<p>Once opened, the disguised executable established persistence using a scheduled task and connected to a C2 server to send system information, receive commands, download additional payloads. The malware used binary string obfuscation and techniques linked to DoNot APT. </p>



<p>The campaign reflects the group’s ongoing espionage efforts using sophisticated infection chains and deception tactics.<gwmw style="display:none;"></gwmw></p>



<p><em>&#8220;The email leveraged diplomatic themes related to defense attaché coordination between Italy and Bangladesh.&#8221; <a href="https://www.trellix.com/blogs/research/from-click-to-compromise-unveiling-the-sophisticated-attack-of-donot-apt-group-on-southern-european-government-entities/">reads Trellix&#8217;s report</a>. &#8220;While the exact body content was not gathered in the findings, the subject line “Italian Defence Attaché Visit to Dhaka, Bangladesh” suggests a lure designed to appear as legitimate diplomatic correspondence that would reasonably contain document attachments or links.&#8221;</em><gwmw style="display:none;"></gwmw><gwmw style="display:none;"></gwmw><gwmw style="display:none;"></gwmw></p>



<figure class="wp-block-image size-large"><a href="https://i0.wp.com/securityaffairs.com/wp-content/uploads/2025/07/image-11.png?ssl=1"><img data-recalc-dims="1" fetchpriority="high" decoding="async" width="1024" height="629" src="https://i0.wp.com/securityaffairs.com/wp-content/uploads/2025/07/image-11.png?resize=1024%2C629&#038;ssl=1" alt="" class="wp-image-179778" srcset="https://i0.wp.com/securityaffairs.com/wp-content/uploads/2025/07/image-11.png?resize=1024%2C629&amp;ssl=1 1024w, https://i0.wp.com/securityaffairs.com/wp-content/uploads/2025/07/image-11.png?resize=300%2C184&amp;ssl=1 300w, https://i0.wp.com/securityaffairs.com/wp-content/uploads/2025/07/image-11.png?resize=768%2C472&amp;ssl=1 768w, https://i0.wp.com/securityaffairs.com/wp-content/uploads/2025/07/image-11.png?w=1335&amp;ssl=1 1335w" sizes="(max-width: 1000px) 100vw, 1000px" /></a></figure>



<p>The malware uses selective obfuscation by packing only critical code sections, hindering static analysis. It minimizes listed imports and loads APIs like <code>LoadLibrary</code> and <code>GetProcAddress</code> at runtime to evade detection. Once executed, it creates a mutex to prevent multiple instances, drops a batch file for persistence via scheduled tasks. The researchers reported that the malicious code uses anti-VM checks.  It gathers system info, encrypts it with AES, and communicates with a C2 server over HTTPS, possibly downloading and executing a second-stage payload (&#8220;socker.dll&#8221;) via further scheduled tasks. The C2 server was inactive during analysis.</p>



<p>The C2 server was inactive during analysis, preventing full observation of the malware’s behavior.</p>



<p>&#8220;The recent targeting of a European foreign affairs ministry highlights their expanding scope and persistent interest in gathering sensitive information, underscoring the need for heightened vigilance and robust cybersecurity measures.&#8221; concludes the report.<gwmw style="display:none;"></gwmw><gwmw style="display:none;"></gwmw></p>
