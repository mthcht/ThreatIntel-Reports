<p>QakBot, also known as QBot, QuackBot, or Pinkslipbot, is a banking trojan malware that has existed for over a decade. In recent years, QakBot has become one of the leading banking trojans around the globe. Its main purpose is to steal banking credentials (e.g., logins, passwords, etc.)</p>



<p>Most of the QBot infections are done by the initial vectors of <a href="https://www.socinvestigation.com/qbot-returns-returns-with-new-ttps-detection-response/" target="_blank" rel="noreferrer noopener" title="XLS documents"><strong>XLS documents</strong></a>. Now, they started using the .lnk files to infect their targeted machines. As usual, this can be done by using spam campaigns or malicious URLs to deliver LNK files to their targets.</p>

<p>Here, the initial vector is the .html file which contains a .zip file with the targeted path of .LNK file. Once the user opens the .LNK file, internal embedded codes will be executed, and it will start its infection chain.</p>



<p>Recent infection can be done by using legitimate applications like PowerShell, CMD, and MSHTA to download the malicious payload files.<br></p>



<h2 class="wp-block-heading"><strong>Why are LNK files being used?</strong></h2>



<p>LNK file is a shortcut or &#8220;link&#8221; used by Windows as a reference to an original file, folder, or application. It contains the shortcut target type, location, and filename as well as the program that opens the target file and an optional shortcut key. The file can be created in Windows by right-clicking a file, folder, or executable program and then selecting create a shortcut.</p>



<p><strong>Also Read: <a href="https://www.socinvestigation.com/latest-iocs-threat-actor-urls-ips-malware-hashes/" target="_blank" rel="noreferrer noopener" title="Latest IOCs – Threat Actor URLs , IP’s &amp; Malware Hashes">Latest IOCs – Threat Actor URLs , IP’s &amp; Malware Hashes</a></strong></p>



<p>In the .lnk files, we can be able to see the target path if it&#8217;s in a shorter range. However, command-line arguments can be up to 4096, so malicious actors can that this advantage and pass on long arguments as they will not be visible in the properties section.</p>

<h2 class="wp-block-heading"><strong>How does QBot LNK work?</strong></h2>



<p>With reference to edge application, Echo &gt; Ping 15.org &gt; %appdata% &gt; curl.exe &gt; .dat &gt; echo &gt; regsvr.</p>



<p>Ping [Packet Internet or Inter-Network Groper ] utility uses the echo request, and echo reply messages within the Internet Control Message Protocol (ICMP), an integral part of any IP network. Here, the ping sends ICMP packets to the destination. Then it waits for the echo reply.</p>
