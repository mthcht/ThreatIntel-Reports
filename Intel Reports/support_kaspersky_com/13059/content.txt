			<h2 class="mrv">The ShadeDecryptor tool for defending against Trojan-Ransom.Win32.Shade</h2>
			
			<div class="fr st_data">
Latest update: September 18, 2023
<span style="padding-left:1em">ID: 13059</span>
</div>
			<div class="cl h1px">&nbsp;</div>
		</div>
		<div class="w_bot">&nbsp;</div>
	</div>


    
    
    
    <div class="wincont_c8">
	<div class="w_cont">
		<h2 class="mrv"></h2>
		<div class="note" id="note_1">


			<div class="fl w-75"><p><p>Do you want to prevent infections? Be safe and private with <a href="/us/buy_kaspersky" target="_blank" title="The Kaspersky website">new Kaspersky applications</a>.</p>
<p>To download the ShadeDecryptor utility, click <strong>Download</strong>.</p></p></div>
				<div class="fr w-25">
					<a href="https://media.kaspersky.com/utilities/VirusUtilities/EN/ShadeDecryptor.zip" class="big-bt" onClick="UAMDownloadBtnTrack(this, '', document.URL )">Download</a>
				</div>




			<div class="cl h1px">&nbsp;</div>
		</div>
	</div>

	<div class="wincont_c3">
		<div class="w_top">&nbsp;</div>
		<div class="w_cont">
                                                                

			
				<div class="wcorn c2_c3">&nbsp;</div>
				
                                                                
			
			
			
			
			                

		                                                <p>The Trojan-Ransom.Win32.Shade malware encrypts the files on the user's computer and makes them inaccessible. Using the ShadeDecryptor tool, you can try to decrypt files with the following extensions: </p>
<ul>
<li>xtbl</li>
<li>breaking_bad</li>
<li>ytbl</li>
<li>heisenberg</li>
<li>better_call_saul</li>
<li>los_pollos</li>
<li>da_vinci_code</li>
<li>magic_software_syndicate</li>
<li>windows10</li>
<li>windows8</li>
<li>no_more_ransom</li>
<li>tyson</li>
<li>crypted000007</li>
<li>crypted000078</li>
<li>dexter</li>
<li>miami_california</li>
<li>rsa3072</li>
<li>decrypt_it</li>
</ul>
<p>The tool searches for the decryption key in its database. If the key is found in the database, the files are decrypted. If the key is not in the databases, the tool sends a request to the server for additional keys. This requires Internet access. </p>
<p>To avoid infection:</p>
<ul>
<li>Download and install <a href="/us/buy_kaspersky" target="_blank" title="The Kaspersky webiste">new Kaspersky applications</a>, which will protect your computer against screen-locking and file-encrypting malware.</li>
<li>Follow the <a href="/us/10952" title="10952 :: How to protect your PC against file-encrypting ransomware">recommendations on how to protect your PC against file-encrypting ransomware</a>.</li>
</ul>

					<h2 class="mrv a_open"><b>How to decrypt files using ShadeDecryptor</b></h2>
					<div id="note_2" class="hidden">
				

		                                                <ol>
<li>Download the <a href="https://media.kaspersky.com/utilities/VirusUtilities/EN/ShadeDecryptor.zip" title="The link to download file">ShadeDecryptor.zip</a> archive and extract the files from it. Use an archiver such as 7-Zip.</li>
<li>Run the ShadeDecryptor.exe file on the infected computer.</li>
</ol>
<p><img title="Starting the ShadeDecryptor utility" alt="Starting the ShadeDecryptor utility." src="https://img.kaspersky.com/kb/en-us/321014_490884_safety101_13059_01.png" /></p>
<ol start="3">
<li>Read through the End User License Agreement carefully. Click <strong>Accept</strong> if you agree to the terms.</li>
<li>Click <strong>Change parameters</strong>.</li>
</ol>
<p><img title="Modifying ShadeDecryptor parameters" alt="Modifying ShadeDecryptor parameters." src="https://img.kaspersky.com/kb/en-us/321014_490885_safety101_13059_02.png" /></p>
<ol start="5">
<li>In the <strong>Objects to scan</strong> section, select the drives you want to scan. To delete encrypted files after they have been decrypted, select the checkbox in the <strong>Additional options</strong> section.</li>
<li>Click <strong>OK</strong>.</li>
</ol>
<p><img title="Configuring scan settings in ShadeDecryptor" alt="Configuring scan settings in ShadeDecryptor." src="https://img.kaspersky.com/kb/en-us/321014_490886_safety101_13059_03.png" /></p>
<ol start="7">
<li>Click <strong>Start scan</strong>.</li>
</ol>
<p><img title="Starting a scan in ShadeDecryptor" alt="Starting a scan in ShadeDecryptor." src="https://img.kaspersky.com/kb/en-us/321014_490887_safety101_13059_04.png" /></p>
<ol start="8">
<li>Specify the path to one of the encrypted files. If the tool is unable to detect the infection ID, it will request the path to the readme.txt file.</li>
</ol>
<p><img title="Example of the readme.txt file for the ShadeDecryptor tool" alt="Example of the readme.txt file for the ShadeDecryptor tool." src="https://img.kaspersky.com/kb/en-us/321014_490888_safety101_13059_05.png" />
<br /></p>
<div start="8">
<p>To view:</p>
<ul start="11">
<li>Information about the scan, click <strong>Details</strong>.</li>
</ul>
<p><strong><img title="Opening information about the ShadeDecryptor scan" alt="Opening information about the ShadeDecryptor scan." src="https://img.kaspersky.com/kb/en-us/321014_490889_safety101_13059_06.png" /></strong></p>
<ul start="11">
<li>A history of all scans performed previously, click <strong>Report</strong> in the top-right corner.</li>
</ul>
<p><img title="Opening scan history in ShadeDecryptor" alt="Opening scan history in ShadeDecryptor." src="https://img.kaspersky.com/kb/en-us/321014_490890_safety101_13059_07.png" />  </p>
