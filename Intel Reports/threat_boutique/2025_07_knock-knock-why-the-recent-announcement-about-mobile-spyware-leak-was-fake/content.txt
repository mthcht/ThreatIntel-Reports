                            <h1 class="title-post" itemprop="name">Knock, knock: Why the recent announcement about mobile spyware leak was fake</h1>
                    <div class="date">
            <span itemprop="datePublished"><a href="https://threat.boutique/archive/2025-07" title="Show all posts made on this month">06 July 2025</a></span>
            - Posted in            <span itemprop="articleSection"><a href="https://threat.boutique/category/threat-analyze">Threat Analyze</a></span> by            <span itemprop="author"><a href="https://threat.boutique/author/b3vSclAuunBMsdt">Jok3r</a></span> -
            <span><a href="https://threat.boutique/2025/07/knock-knock-why-the-recent-announcement-about-mobile-spyware-leak-was-fake" rel="permalink">Permalink</a></span>
        </div>
                    <div class="featured-image">
                <a href="https://threat.boutique/2025/07/knock-knock-why-the-recent-announcement-about-mobile-spyware-leak-was-fake"><img src="https://threat.boutique/content/images/20250706221056-malw.png" alt="Knock, knock: Why the recent announcement about mobile spyware leak was fake"/></a>
            </div>
                                        <div class="post-body post-1751825445" itemprop="articleBody">
            <p>Background:
Recently, a Telegram account announced the publication of source codes related to a mobile spyware. Before drawing any final conclusions, it is important to understand that this announcement was made in the context of various geopolitical conflicts. In addition, we are facing two challenges: identifying the source and conducting package research.
<img src="https://threat.boutique/content/images/20250706221523-er.png" alt="enter image description here" /></p>

<p>Research:
The package contained some files, including PDFs and source code. Now, let's take a look at the PDF.
<img src="https://threat.boutique/content/images/20250706221709-er1.png" alt="enter image description here" />
The PDF contains a leaked document related to spyware, which is quite old and has been seen previously.
<img src="https://threat.boutique/content/images/20250706221945-er2.png" alt="enter image description here" />
The next file is a 7z archive named "pegasus n.s.o". Upon review, we can see that this tool is open-source and freely available. It has no connection with the actual mobile spyware vendorâ€™s product.
<img src="https://threat.boutique/content/images/20250706222354-er3.png" alt="enter image description here" />
Other files were related to Windows products associated with open-source RATs.</p>

<p><img src="https://threat.boutique/content/images/20250706223006-er4.png" alt="enter image description here" /></p>

<p>Another file contained a malicious (malwarized) file, but it has been freely available since 2018.
<img src="https://threat.boutique/content/images/20250706223320-er5.png" alt="enter image description here" /></p>

<p>Conclusion:
Based on the current analysis, the file that was impersonated as a new leak appears to be quite old. The confidence in claiming that it is a unique leak is low; it is likely a false alarm.</p>
