                                        <div class="post-body post-1755012363" itemprop="articleBody">
            <p>Background:
Recently, some unknown specialists uploaded several dumps from servers that, according to their claims, are related to Kimsuky (State sponsored) systems. As cybersecurity defenders, we need to analyze all these dumps to understand the threat actorâ€™s digital weapons and exploits. Based on this analysis, we can implement appropriate defense layers to protect against such attacks.</p>

<p>Toolset used by that threat actors:
This part is particularly interesting because, if you have dumps of all the systems, as a cybersecurity professional you can first examine their bash history to understand user activity. Based on this search, I have identified the toolsets that the threat actors have used or are currently using.</p>

<ul>
<li>BetterOCR :
This tools is helping to translate pdf and ocr components into different  languages</li>
</ul>

<p><img src="https://threat.boutique/content/images/20250812192749-1.png" alt="enter image description here" />
 - IntelliJIdea2018.3 :
Ide for ongoing work
 - minibeacon :
http(s) beacon for Cobalt Strike
<img src="https://threat.boutique/content/images/20250812193143-3.png" alt="enter image description here" />
 -TitanLdr :
 Loader written in C and assembly for Cobalt Strike. Redirects DNS Beacon over DoH
<img src="https://threat.boutique/content/images/20250812193409-4.png" alt="enter image description here" />
 - proxyres : C lang library for   proxy
 <img src="https://threat.boutique/content/images/20250812193617-5.png" alt="enter image description here" />
 - Custom-PE-Packer : Tool to evade from AV detection
 - blacklotus : A well known UEFI bootkit
 - RATandC2 : C2 and Rat coplex tool
<img src="https://threat.boutique/content/images/20250812194923-6.png" alt="enter image description here" />
 - Cobalt strike component
<img src="https://threat.boutique/content/images/20250812195039-7.png" alt="enter image description here" />
 - Muraena : tool for phishing and post phishing proxy activities
<img src="https://threat.boutique/content/images/20250812195309-8.png" alt="enter image description here" />
 - Mettle &amp; Winrar old exploit
<img src="https://threat.boutique/content/images/20250812200042-10.png" alt="enter image description here" /></p>

<p>To protect environment from such threat need to  execute noted activity</p>

<ul>
<li>EDR/XDR installation on all server</li>
<li>Enable Secure Boot  and force firmware update on all machines</li>
<li>Implement IDS/IPS on  network layer</li>
<li>Phishing &amp; Web Protection &amp; Employees awareness training</li>
<li>Threat hunting</li>
</ul>
