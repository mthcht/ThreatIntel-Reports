    <div class='cylance-blog-container'><h2>Executive Overview</h2> <p>Neshta is an older file infector that is still prevalent in the wild. It was initially observed in 2003 and has been previously associated with BlackPOS malware. It prepends malicious code to infected files. This threat is commonly introduced into an environment through unintentional downloading or by other malware. It infects Windows executable files and may attack network shares and removable storage devices.</p> <p>In 2018 Neshta predominantly targeted the manufacturing industry, but attacked the finance, consumer goods, and energy sectors as well. To achieve persistence Neshta renames itself to svchost.com then modifies the registry so it runs each time an .exe file is launched. This threat is known to collect system information and use POST requests to exfiltrate data to attacker-controlled servers. The Neshta binaries used in our analysis did not demonstrate the data exfiltration behaviour or functionality.</p> <h2>Technical Analysis</h2> <p>This section describes the symptoms of a Neshta infection. We picked samples of the Neshta virus uploaded to VirusTotal in 2007, 2008, and 2019.<br> <br> We analyzed files with these SHA-256 hashes:</p> <ul> <li>29fd307edb4cfa4400a586d38116a90ce91233a3fc277de1cab7890e681c409a</li> <li>980bac6c9afe8efc9c6fe459a5f77213b0d8524eb00de82437288eb96138b9a2</li> <li>539452719c057f59238e123c80a0a10a0b577c4d8af7a5447903955e6cf7aa3d</li> <li>a4d0865565180988c3d9dbf5ce35b7c17bac6458ef234cfed82b4664116851f2</li> <li>46200c11811058e6d1173a2279213d0b7ccde611590e427b3b28c0f684192d00</li> <li>c965f9503353ecd6971466d32c1ad2083a5475ce64aadc0b99ac13e2d2c31b75</li> </ul> <h2>Static File Analysis</h2> <p>Neshta’s code is compiled by Boland Delphi 4.0. The file size is usually 41,472 bytes.</p> <p>As any Delphi binary, Neshta has four writable sections (DATA, BSS, .idata, and .tls) and three sharable sections (.rdata, .reloc, and .rsrc):</p> <p style="text-align: center;"><img src="/content/dam/blogs-blackberry-com/images/blogs/2019/10/fig1-neshta.png"><br> </p> <p style="text-align: right;"><i>Figure 1: Section header features</i></p> <p>In addition, Neshta code displays interesting fingerprint strings - see Figure 2, below:<br> <br> “<i>Delphi-the best. F*** off all the rest. Neshta 1.0 Made in Belarus. Прывiтанне усiм ~цiкавым~ беларус_кiм дзяучатам. Аляксандр Рыгоравiч, вам таксама :) Восень- кепская пара... Алiварыя - лепшае пiва! Best regards 2 Tommy Salo. [Nov-2005] yours [Dziadulja Apanas]”</i></p> <p>The Belarusian strings say: <i>“ Hello everyone ~ ~ interesting belarus_kim girls. Alexander G., you too :) osen- bad couple ... Alivaryya - the best beer!”</i></p> <p style="text-align: center;"><img src="/content/dam/blogs-blackberry-com/images/blogs/2019/10/fig2-neshta.png"></p> <p style="text-align: right;"><i>Figure 2: Neshta’s fingerprint strings</i></p> <h2>File Infection</h2> <p>Neshta’s main feature is its file infector which searches local drives for .exe files. Neshta targets “.exe” files making exceptions for the ones that contain any of the following strings in their short path:</p> <ul> <li>%Temp%</li> <li>%SystemRoot%&nbsp; (usually C:\Windows)</li> <li>\PROGRA~1\</li> </ul> <p>The infection flow summary is described below and shown in Figure 3.</p> <p>Neshta:</p> <ol> <li>Reads 41,472 (0xA200) bytes from the beginning of target original file.</li> <li>Creates two sections and allocates memory with the attribute of PAGE_READWRITE on the original file’s beginning and bottom.</li> <li>Inserts its malicious header and code at the beginning of the original file. The written data is 41,472 bytes.</li> <li>Writes the encoded original header and code to the file, which is 41,472 bytes in size.</li> </ol> <p>These actions enable the malicious code to launch as soon as the infected file is executed:</p> <p style="text-align: center;"><img src="/content/dam/blogs-blackberry-com/images/blogs/2019/10/fig3-neshta.png"></p> <p style="text-align: right;"><i>Figure 3: File Infection</i></p> <p>When the infected file is executed, the original program is dropped into %Temp%\3582-490\&lt;filename&gt; and run by the WinExec API.</p> <h2>Persistence</h2> <p>Neshta drops itself to C:\Windows\svchost.com and installs itself into the registry using the following parameters:</p> <p style="margin-left: 40.0px;"><b>Registry key:</b> HKLM\SOFTWARE\Classes\exefile\shell\open\command<br> <b>Registry value: </b>(Default)<br> <b>Value:</b> %SystemRoot%\svchost.com &quot;%1&quot; %*</p> <p>This registry change directs the system to run Neshta each time an .exe file is launched. The &quot;%1&quot; %* points to the launched .exe file. In addition, Neshta creates a named mutex to check for the existence of another running instance:&nbsp;</p> <p><i>MutexPolesskayaGlush*.*&lt;0x90&gt;svchost.com&lt;0x90&gt;exefile\shell\open\command‹À &quot;%1&quot; %*œ‘@</i></p> <p>Another dropped file is “directx.sys” which is sent to %SystemRoot%. This is a text file (not a kernel driver) which contains the path of the last infected file to run. It is updated every time an infected file is executed.</p> <h2>BlackBerry Cylance Stops Neshta</h2> <p>BlackBerry Cylance uses artificial intelligence-based agents trained for threat detection on millions of both safe and unsafe files. Our automated security agents block Neshta based on countless file attributes and malicious behaviors instead of relying on a specific file signature. BlackBerry Cylance, which offers a <a href="https://threatvector.cylance.com/en_us/home/cylance-vs-future-threats-the-predictive-advantage.html" target="_blank">predictive advantage</a> over zero-day threats, is trained on and effective against both new and legacy cyberattacks. For more information, visit &nbsp;<a href="https://www.cylance.com/" target="_blank">https://www.cylance.com</a>.</p> <h2>Appendix</h2> <h2>Indicators of Compromise (IOCs)</h2> <ul> <li>Hashes<br> o&nbsp;&nbsp; 29fd307edb4cfa4400a586d38116a90ce91233a3fc277de1cab7890e681c409a<br> o&nbsp;&nbsp; 980bac6c9afe8efc9c6fe459a5f77213b0d8524eb00de82437288eb96138b9a2<br> o&nbsp;&nbsp; 539452719c057f59238e123c80a0a10a0b577c4d8af7a5447903955e6cf7aa3d<br> o&nbsp;&nbsp; a4d0865565180988c3d9dbf5ce35b7c17bac6458ef234cfed82b4664116851f2<br> o&nbsp;&nbsp; 46200c11811058e6d1173a2279213d0b7ccde611590e427b3b28c0f684192d00<br> o&nbsp;&nbsp; c965f9503353ecd6971466d32c1ad2083a5475ce64aadc0b99ac13e2d2c31b75<br> <br> </li> <li>Filenames<br> o&nbsp;&nbsp; %SystemRoot%\svchost.com<br> o&nbsp;&nbsp; %SystemRoot%\directx.sys<br> o&nbsp;&nbsp; %Temp%\tmp5023.tmp<br> <br> </li> <li>C2s/IPs</li> <li>Mutexes<br> o&nbsp;&nbsp; MutexPolesskayaGlush*.*&lt;0x90&gt;svchost.com&lt;0x90&gt;exefile\shell\open\command‹À &quot;%1&quot; %*œ‘@<br> <br> </li> <li>Interesting strings<br> o&nbsp;&nbsp; Delphi-the best. F**k off all the rest. Neshta 1.0 Made in Belarus. Прывiтанне усiм ~цiкавым~ беларус_кiм дзяучатам. Аляксандр Рыгоравiч, вам таксама :) Восень- кепская пара... Алiварыя - лепшае пiва!
