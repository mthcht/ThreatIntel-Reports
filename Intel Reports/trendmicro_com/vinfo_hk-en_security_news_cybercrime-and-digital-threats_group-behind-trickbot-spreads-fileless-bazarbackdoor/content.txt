        <div><img src="https://documents.trendmicro.com/images/TEx/articles/20180405232356493-31-uzfkrra-800.jpg" alt="" width="300" height="225" style="float: right;">A new campaign is propagating a new malware named “BazarBackdoor,” a fileless backdoor reportedly created by the same threat actors behind TrickBot, as reported by <a href="https://www.bleepingcomputer.com/news/security/bazarbackdoor-trickbot-gang-s-new-stealthy-network-hacking-malware/">BleepingComputer</a>. The conclusion is drawn due to similarities in code, crypters, and infrastructure between the two malware variants.<br><br></div>
<div>The social engineering attacks that were used to spread the backdoor leverage topics such as customer complaints, <a href="https://www.trendmicro.com/vinfo/us/security/news/cybercrime-and-digital-threats/coronavirus-used-in-spam-malware-file-names-and-malicious-domains">Covid-19-themed</a> payroll reports, and employee termination lists for the emails they send out. The messages have links to Google Docs files. Once the users click the links, they will be redirected to a landing page. The pages state that the Word Document, Excel Spreadsheet, or PDF cannot be properly viewed. It then instructs the user to click on a link to open the file.<br><br></div>
<div>Clicking on the link downloads an executable that masquerades through icons and names associated with the mentioned file types. For instance, the supposed customer complaint document will be downloaded as Preview.PDF.exe, which uses the PDF icon. Since the file extension is hidden by default, the file will convincingly appear as a PDF file.<br><br></div>
<div>The disguised executable serves as the loader for the backdoor. After launching the file, the loader sleeps for some time, then connects to command and control (C&C) servers to check-in and download the payload. The payload will then be injected filelessly into C:\Windows\system32\svchost.exe through <a href="https://blog.trendmicro.com/trendlabs-security-intelligence/almost-hollow-and-innocent-monero-miner-remains-undetected-via-process-hollowing/">process hollowing</a> and <a href="https://www.trendmicro.com/vinfo/us/security/news/cybercrime-and-digital-threats/synack-ransomware-leverages-process-doppelg-nging-for-evasion-and-infection">process doppelgänging</a> techniques. The backdoor will be installed on the computer.<br><br></div>
<div>This sets a scheduled task that launches the loader every time the user logs into Windows, which makes way for new versions of the backdoor to be downloaded and injected into svchost.exe. Security researchers <a href="https://twitter.com/VK_Intel">Vitali Kremez</a> and James revealed that this malware was most likely created by the threat actors behind TrickBot trojan. This is because both malware types use the same crypter and email chain deliverables. Both malware also utilize the Emercoin DNS resolution service for C&C server communication.</div>
<div> </div>
<h3><strong>Defense against fileless threats</strong></h3>
<div>Fileless threats are stealthy and difficult to detect because they take advantage of existing applications to infiltrate and attack systems. However, users can still defend against these malware types by adhering to the following best practices:</div>
<div><br>[Related: <a href="https://www.trendmicro.com/vinfo/us/security/news/security-technology/risks-under-the-radar-understanding-fileless-threats">Risks Under the Radar: Understanding Fileless Threats</a>]</div>
<div style="text-align: justify;">
<ul>
<li>Secure possible entry points. Malicious sites, <a href="https://www.trendmicro.com/vinfo/us/security/news/cybercrime-and-digital-threats/infosec-guide-email-threats">spam</a>, and third-party components like browser plug-ins can all be sources of fileless malware. Be cautious when downloading attachments and other files, and never click links from unfamiliar sources.</li>
<li>Reboot device and change passwords. In case of infection, users can stop fileless attacks that do  not employ persistence techniques by restarting the device. As an extra precaution, users should also change their passwords.</li>
<li>Utilize behavior monitoring and analysis. These can detect and block malicious behaviors and routines associated with malware, stopping threats before they can reach the system.</li>
</ul>
</div>
<div style="text-align: justify;">To further secure the system, the following security solutions are recommended:</div>
<div style="text-align: justify;">
<ul>
<li><a href="https://www.trendmicro.com/en_us/business/products/user-protection/sps/endpoint.html">Trend Micro Apex One™</a>– Employs behavior analysis to protect systems against malicious scripts, injection, ransomware, and memory and browser attacks related to fileless threats.</li>
<li><a href="https://www.trendmicro.com/en_us/business/products/user-protection/sps/endpoint.html">Trend Micro Apex One Endpoint Sensor</a> – Through Endpoint Detection and Response (EDR) and X Detection and Response (XDR), monitors events and processes that trigger malicious activity.</li>
<li><a href="https://www.trendmicro.com/en_us/small-business/worry-free-services.html">Trend Micro Worry-Free Services</a> – Utilizes behavior monitoring to detect script-based, fileless threats, preventing malware from entering the system.</li>
</ul>
</div>
