    <section class="articleContent"> 


       

        <p><em><img src="https://documents.trendmicro.com/images/TEx/articles/20180416041606314-312-m0y1bwc-800.jpg" alt="" width="300" height="225" style="float: right;" />Additional insights by Monte De Jesus, Mohammed Malubay, and Alyssa Christelle Ramos<br /><br />Updated on July 23, 2020 3 AM EDT with added data on new ransomware families.</em></p>
<p>This past couple of months, <a href="https://www.trendmicro.com/vinfo/US/security/definition/ransomware">ransomware</a> has remained a formidable threat as new families, techniques, and targets continue emerging at every turn. Recently, we witnessed the rise of new ransomware family Avaddon. We also examined techniques utilized by some ransomware variants and the industries affected by these attacks. Additionally, we included our latest figures about ransomware families with the most detections, new ransomware families, and the most affected industries and segments.</p>
<h3><strong>Avaddon ransomware</strong></h3>
<p>The new ransomware called Avaddon (detected by Trend Micro as <a href="https://www.trendmicro.com/vinfo/us/threat-encyclopedia/malware/Ransom.Win32.AVADDON.YJAF-A">Ransom.Win32.AVADDON.YJAF-A</a>) has been observed at large.  A trojan (detected by Trend Micro as <a href="https://www.trendmicro.com/vinfo/us/threat-encyclopedia/malware/Trojan.JS.AVADDON.YJAF-A">Trojan.JS.AVADDON.YJAF-A</a>) downloads the ransomware from malicious sites and runs them on the system. This has been reported in a series of <a href="https://twitter.com/tmmalanalyst/status/1270197110768406528">twitter posts by TMMalAnalyst</a>.</p>
<p>The ransomware is propagated through emails with an attachment named IMG{6 random number}.jpg.js.zip that contains a JavaScript file named IMG{6 random number}.jpg.js.<br /><br /><img src="https://documents.trendmicro.com/images/TEx/articles/ransomware report 1.png" alt="" width="736" height="516" style="display: block; margin-left: auto; margin-right: auto;" /></p>
<p style="text-align: center;">Figure 1. Sample email for Avaddon campaign</p>
<p>As seen in the preceding figure, the email body contains a single smiley. The emails for the Avaddon campaign also  follow the footsteps of past malware campaigns that use particular subjects to spark the curiosity of the users,  thus prompting them to open the message and download the attachment. Most of these emails have photo-related subjects, which might be particularly enticing for users at a time when gadgets with built-in cameras have now become widely available:</p>
<ul>
<li>Look at this photo!</li>
<li>Photo just for you</li>
<li>You look good here</li>
<li>I love this photo</li>
<li>I like this photo</li>
<li>Is this your photo?</li>
<li>Is this you?</li>
<li>My favourite photo</li>
<li>You like this photo?</li>
</ul>
<p>After the attachment is downloaded and ran, it uses a PowerShell command and the BITSAdmin command-line tool to download and run the ransomware payload. After this, the affected users will see that the ransomware has encrypted the files and appended them with the .avdn file extension. Users will see that their system desktop’s wallpaper has been automatically changed to an image that states that “all your files have been encrypted” and refers to the ransom note: “Instruction 270015-readme.html” (following the {Encrypted Directory}\{random numbers}-readme.html format):<br /><br /><img src="https://documents.trendmicro.com/images/TEx/articles/ransomware report 2 background.jpg" alt="" width="540" height="251" style="display: block; margin-left: auto; margin-right: auto;" /></p>
<p style="text-align: center;">Figure 2. User’s wallpaper as modified by the Avaddon attack</p>
<p>The ransom note gives instructions on how the affected user can recover the encrypted files.<br /><br /><img src="https://documents.trendmicro.com/images/TEx/articles/ransomware report 3 ransom note.jpg" alt="" width="540" height="607" style="display: block; margin-left: auto; margin-right: auto;" /></p>
<p style="text-align: center;">Figure 3. Avaddon ransom note</p>
<p>This ransomware encrypts files found in the following folders:</p>
<ul>
<li>Program Files\Microsoft\Exchange Server</li>
<li>Program Files (x86)\Microsoft\Exchange Server</li>
<li>Program Files\Microsoft SQL Server</li>
<li>Program Files (x86)\Microsoft SQL Server</li>
</ul>
<p>It adds the following processes that deletes backup copies of the system, making it difficult to restore:</p>
<ul>
<li>wmic.exe SHADOWCOPY /nointeractive</li>
<li>wbadmin DELETE SYSTEMSTATEBACKUP</li>
<li>wbadmin DELETE SYSTEMSTATEBACKUP -deleteOldest</li>
<li>bcdedit.exe /set {default} recoveryenabled No</li>
<li>bcdedit.exe /set {default} bootstatuspolicy ignoreallfailures</li>
<li>vssadmin.exe Delete Shadows /All /Quiet</li>
</ul>
<p>It terminates services and processes, many of which are related to scanning, storing and retrieving files, and scheduling tasks. Below are some examples:</p>
<p>Terminated services:</p>
<ul>
<li>ccEvtMgr</li>
<li>ccSetMgr</li>
<li>Culserver</li>
<li>dbeng8</li>
<li>dbsrv12</li>
<li>DefWatch</li>
<li>Intuit.QuickBooks.FCS</li>
<li>msmdsrv</li>
<li>QBCFMonitorService</li>
<li>QBIDPService</li>
</ul>
<p>Terminated processes:</p>
<ul>
<li>360doctor.exe</li>
<li>360se.exe</li>
<li>axlbridge.exe</li>
<li>BCFMonitorService.exe</li>
<li>Culture.exe</li>
<li>Defwatch.exe</li>
<li>fdhost.exe</li>
<li>fdlauncher.exe</li>
<li>GDscan.exe</li>
<li>httpd.exe</li>
</ul>
<p>It terminates itself if the Windows Locale ID is equal to the following:</p>
<ul>
<li>419 = Russian</li>
<li>422 = Ukrainian</li>
</ul>
<p>It terminates itself if machine is set to the following keyboard layout language:</p>
<ul>
<li>419 = Russian</li>
<li>485 = Yakut (Russia)</li>
<li>444 = Tatar</li>
<li>422 = Ukrainian</li>
</ul>
<p>It is worth mentioning that the technique of avoiding systems from particular countries has similarly been observed in <a href="https://www.2-spyware.com/remove-medusalocker-ransomware.html">MedusaLocker ransomware</a> campaigns.</p>
<p>For a full list of processes and services and for more details about the ransomware, please refer to our <a href="https://www.trendmicro.com/vinfo/us/threat-encyclopedia/malware/Ransom.Win32.AVADDON.YJAF-A">report</a>.</p>
<h3><strong>New techniques spotted</strong></h3>
<p>In recent months, there have also been updates on the techniques used by some ransomware variants. For example, Netwalker ransomware can now be run <a href="https://blog.trendmicro.com/trendlabs-security-intelligence/netwalker-fileless-ransomware-injected-via-reflective-loading/">filelessly through reflective dynamic-link library (DLL)</a> injection (aka reflective DLL loading). This technique injects the DLL from memory rather than from disk.  Although the technique itself is not novel (it has been previously used to deploy <a href="https://blog.trendmicro.com/trendlabs-security-intelligence/targeted-ransomware-attack-hits-taiwanese-organizations/">ColdLock</a> ransomware), its use by Netwalker is new.</p>
<p>Another notable development is Ragnar Locker’s deployment of virtual machines to evade detection by antivirus software. According to <a href="https://news.sophos.com/en-us/2020/05/21/ragnar-locker-ransomware-deploys-virtual-machine-to-dodge-security/">Sophos</a>, this attack vector has never been used with any ransomware type before. In the past, Ragnar Locker exploited managed service providers or attacks on Windows Remote Desktop Protocol (RDP) connections.</p>
<h3><strong>Manufacturing, logistics, and energy sectors as targets</strong></h3>
<p>Ransomware varieties have been used to target several companies under the manufacturing, logistics, and energy sectors in the past months. A variant of <a href="https://securityboulevard.com/2020/06/ekans-strikes-again-honda-and-enel-taken-down-by-ransomware/">Ekans ransomware</a> (detected by Trend Micro as Ransom.Win32.EKANS.D) has been wielded in targeted attacks against manufacturing companies. As observed by <a href="https://www.dragos.com/blog/industry-news/ekans-ransomware-and-ics-operations/">Dragos</a>, there is a particular level of intentionality that is evident in the industrial processes terminated in past Ekans attacks, making them a threat that organizations with industrial control systems (ICS) should keep an eye out for.</p>
<p><a href="https://www.trendmicro.com/vinfo/us/security/news/cybercrime-and-digital-threats/updated-analysis-on-nefilim-ransomware-s-behavior">Nefilim</a><span>,</span> a ransomware that follows the recent trend of ransomware types that not only encrypt files but also steal data, has been witnessed to attack logistics companies. Investigations into these attacks have led us to uncover more about the recently discovered ransomware’s behavior, particularly with regard to its data theft capabilities. We found out that this data theft begins weeks or even months before the ransomware is deployed, and that the attacks use several tools (both malicious and non-malicious) to deploy processes and move through the network.</p>
<p>In related news, operators behind Sodinokibi published, on a Tor webpage, 1,280 files of what they claim to be the passport details and other documents of staff members of an electric service provider. A few weeks before this, the ransomware attack <a href="https://www.theregister.com/2020/06/01/elexon_ransomware_was_revil_sodinokibi/">struck the company</a>, thereby interrupting their operations.</p>
<p>On the other hand, another ransomware which we dubbed as ColdLock (detected by Trend Micro as <span>Ransom.MSIL.COLDLOCK.YPAE-A) </span>targeted a region, rather than just a particular industry. Specifically, it launched attacks on <a href="https://blog.trendmicro.com/trendlabs-security-intelligence/targeted-ransomware-attack-hits-taiwanese-organizations/">Taiwanese organizations</a>, aiming to target databases and email servers for encryption.</p>
<h3><strong>Ransomware figures for May</strong></h3>
<p>For May, WannaCry emerged as the top ransomware family with 15,496 detections. WannaCry’s retention of the highest number of detections can be attributed to its worm component and its operators’ persistence in trying to propagate the malware regularly.  We foresee that WannaCry will continue having such a high number of detections until either a new, massive ransomware comes into being, or the sources for WannaCry are found and removed.</p>
<p>Trailing behind are Locky with 1,532 detections and Cerber with 392 detections. Indeed, these ransomware families have consistently been on the top three since January of this year. They were also on the top three for last year’s total ransomware detections.</p>
<p style="text-align: center;"><img src="https://documents.trendmicro.com/images/TEx/articles/ransomware report 4 families.jpg" alt="" width="972" height="610" /></p>
<p style="text-align: center;">Figure 4. Ransomware families with the most detections (May 2020)</p>
<p>In the same month, the industries with the most detections were government (1,870), manufacturing (1,599), and healthcare (1,217).</p>
<p style="text-align: center;"><img src="https://documents.trendmicro.com/images/TEx/articles/ransomware report 5 industries.jpg" alt="" width="970" height="583" /></p>
<p style="text-align: center;">Figure 5. Top industries for ransomware detections (May 2020)</p>
<p>For segments, enterprise had the highest number of detections with over 18,000. Meanwhile, detections in the consumer segment numbered over 4,000, compared with over 1,000 detections in small and medium-sized businesses (SMB).</p>
<p style="text-align: center;"><img src="https://documents.trendmicro.com/images/TEx/articles/ransomware report 6 segments.jpg" alt="" /><img src="https://documents.trendmicro.com/images/TEx/articles/ransomware report 6 segment.jpg" alt="" width="947" height="563" /></p>
<p style="text-align: center;">Figure 6. Top segments for ransomware detections (May 2020)</p>
<p>As for ransomware families, five new ones were detected in May, including the aforementioned ransomware ColdLock. One of these new families is BlueCheeser (detected by Trend Micro as Ransom.MSIL.BLUECHEESER.A), a ransomware family that appends encrypted files <a href="https://www.pcrisk.com/removal-guides/16965-bluecheeser-ransomware">with the .himr extension</a> and instructs affected users to pay US$400 to decrypt files.</p>
<p>Another is CoronaLock (detected by Trend Micro as Ransom.Win32.CORONALOCK.A), <a href="https://www.2-spyware.com/remove-corona-lock-ransomware.html">also known as CovidWorldCry</a>. This ransomware, propagated through coronavirus-themed spam, renames encrypted files with .corona.lock extension. A different ransomware family named PonyFinal (detected by Trend Micro as Ransom.Java.PONYFINAL.A) is a <a href="https://threatpost.com/ponyfinal-ransomware-enterprise-servers/156083/">Java-based</a>, human-operated ransomware that targets Microsoft systems. Lastly, GonnaCry (detected by Trend Micro as Ransom.Linux.GONNACRY.A) is a ransomware that targets Linux systems. Compared with detections in April, the number of new ransomware families detected has decreased.</p>
<p><img src="https://documents.trendmicro.com/images/TEx/articles/Ransomware Bar chart 2-01-v2.jpg" alt="" width="859" height="823" style="display: block; margin-left: auto; margin-right: auto;" /></p>
<p style="text-align: center;">Figure 7. Number of new ransomware families (January to May 2020)</p>
<h3><strong>Robust defense against ransomware</strong></h3>
<p>Interrupted operations, lost data, and the publication of confidential company data are some of the ways that a ransomware attack can put a company at risk. However, companies can still find ways to protect their organizations from these attacks.</p>
<p>Here are some  of the <a href="https://www.trendmicro.com/vinfo/us/security/news/cybercrime-and-digital-threats/best-practices-ransomware">best practices</a> for users to protect systems from ransomware:</p>
<ul>
<li>Back up files using the <a href="https://blog.trendmicro.com/trendlabs-security-intelligence/world-backup-day-the-3-2-1-rule/?_ga=2.69332048.1941080928.1587017369-534757267.1571299636">3-2-1 rule</a>. This rule involves regularly creating three backups in two different formats while storing one copy off-site.</li>
<li>Periodically patch and update applications and software. This ensures that vulnerabilities are addressed. For <a href="https://www.trendmicro.com/vinfo/us/security/definition/zero-day-vulnerability">zero-day vulnerabilities</a>, deploy <a href="https://www.trendmicro.com/vinfo/us/security/definition/zero-day-vulnerability">virtual patching</a>.</li>
<li>Enable sandbox analysis. Through this, malicious files can be run in an isolated environment. Therefore, these files can be monitored without putting the system at risk.</li>
<li>Enable advanced detection capabilities for new ransomware families like machine learning or behavior monitoring technologies within your solutions.</li>
</ul>
